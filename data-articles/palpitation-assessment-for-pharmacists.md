# 🩺 動悸が突然始まったら？薬剤師が確認したいポイント

<div style="background-color: #f0f9ff; border-left: 4px solid #0284c7; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  薬局の窓口で、継続処方の患者さんから「先生、昨日からなんだか急に胸がドキドキして……」と相談を受けました。私はてっきり「お薬の副作用かな？」と思い、直近で追加された薬剤の添付文書ばかりを確認していました。<br><br>
  しかし、よくよく話を聞くと、その患者さんは「立っているのも辛い」「冷や汗が出る」と訴えていたのです。すぐに医師に連絡を取ると「それはすぐに循環器へ行って！」との指示。もし私が「様子を見ましょう」と返していたらと思うと、背筋が凍る思いをしました。動悸＝副作用やストレスと決めつけることの危険性を痛感したエピソードです。
</div>

動悸は、薬局業務において遭遇する訴えの一つですが、背景には「緊急性の高い疾患」が隠れていることもあります。今回は、<strong>「動悸の聞き取りで、薬剤師がどこに注目すべきか」</strong>をテーマに整理します。

## 🔍 1. まずはココを確認！動悸の「聞き取り4大項目」

患者さんから動悸の訴えがあった際、まずは以下の4点を具体的に聞き出しましょう。これにより、緊急度や薬学的介入の必要性が見えてきます。

| 項目 | 確認したい内容 | 判断のポイント |
| :--- | :--- | :--- |
| <strong>リズム</strong> | ドキッとするのか、速いのか | 期外収縮か、頻拍か |
| <strong>持続時間</strong> | 数秒か、ずっと続くのか | 一過性か、持続的か |
| <strong>随伴症状</strong> | 息切れ、胸痛、冷や汗、ふらつき | <strong>レッドフラッグ（緊急）</strong>の可能性 |
| <strong>誘因</strong> | 運動中か、安静時か、薬剤服用後か | 心臓への負荷・薬剤性の疑い |

  以下の症状が一つでもあれば、ただちに医師への連絡・相談、または救急外来への受診を推奨してください。<br>
  ・胸の痛み、圧迫感を伴う<br>
  ・強い息切れ、呼吸困難がある<br>
  ・意識が遠のく感じ、失神の既往<br>
  ・冷や汗が出るほどの強い不快感
</div>

## 💊 2. 薬剤性の動悸を疑うケース

薬剤師として特に意識したいのが、服用中の薬剤による副作用です。以下の薬剤を服用中の患者さんが動悸を訴えた場合は、副作用の可能性を考慮します。

### 頻度の高い薬剤性の動悸
- <strong>交感神経刺激薬</strong>: 気管支拡張薬（β2刺激薬など）、感冒薬（プソイドエフェドリン等）
- <strong>甲状腺ホルモン製剤</strong>: 用量過多による心拍数増加
- <strong>利尿薬</strong>: 低カリウム血症に伴う不整脈
- <strong>その他</strong>: 一部の抗うつ薬、漢方薬（麻黄を含むもの）

## ⚡ 3. 不整脈の種類と特徴的な症状

患者さんの「ドキドキする」という表現は様々です。言葉の端々からどのような状態を疑うか、臨床的なヒントを整理しました。

## 🩺 4. 薬剤師ができる「次のアクション」

患者さんから動悸の相談を受けたとき、薬剤師には以下の対応が求められます。

1. <strong>お薬手帳と持病の確認</strong>: 循環器疾患の既往や、抗凝固薬等の使用状況を即座に確認。
2. <strong>直近の変更点の確認</strong>: 服用開始や増量した薬剤がないか。
3. <strong>疑義照会・受診勧奨</strong>: レッドフラッグがあれば即時対応。そうでなくても、循環器内科への受診を提案し、必要に応じて紹介状の準備や医師への情報共有（疑義照会・報告）を行います。
4. <strong>バイタルサイン測定の活用</strong>: 薬局に血圧計やパルスオキシメーターがあれば、必要に応じて測定を促すのも一つの手段です。

## 🌿 5. まとめ：薬剤師の役割は「リスクの仕分け」

動悸という症状は、単なるストレスから重篤な心疾患まで非常に幅が広いです。
私たち薬剤師に求められているのは、「すぐに受診すべき状態か（緊急性）」を判断し、<strong>「薬剤が関与している可能性」を常に冷静に評価すること</strong>です。

「単なる疲れだろう」と自己判断せず、患者さんの訴えをフラットに聞き取り、専門家として適切なトリアージを行うことが、患者さんの命を守る第一歩になります。もし迷ったときは、独断で経過観察とせず、処方医へ「動悸を訴えている」旨をシンプルかつ迅速に伝えましょう。
