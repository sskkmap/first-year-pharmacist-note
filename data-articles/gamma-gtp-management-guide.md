# γ-GTPが上がったら？薬だけが原因ではない？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ノート：その「γ-GTP上昇」は薬のせい？】</strong><br>
  ある日、定期受診の患者さんから「先生に『γ-GTPが高いから薬を少し減らそうか』と言われたんだけど、この薬（脂質異常症薬）が悪いのかな？」と不安げに相談されました。カルテを確認すると、AST・ALTは正常で、γ-GTPのみが緩やかに上昇している状態でした。<br><br>
  私は反射的に「薬剤性」を疑いましたが、詳しく問診をしてみると、ここ1ヶ月ほど「仕事が忙しく、晩酌の量が増えていた」という事実が判明。結局、薬の変更ではなく、生活習慣の改善で数値は落ち着きました。薬のせいだと決めつけて疑義照会を出す前に、もっと広い視野で確認すべきことがあると学んだ苦い経験です。
</div>

健康診断や血液検査の結果を見て「γ-GTPが高いから、飲んでいる薬のせいかな？」と安易に結論づけていませんか？薬剤師として、検査値の変動を正しく読み解き、患者さんに適切な助言をするために知っておくべき知識をまとめました。

---

## 🔬 1. γ-GTP上昇を見たら、まず確認すべき3つの視点

γ-GTPは肝胆道系酵素の中でも、特に「アルコール」や「胆道系の閉塞」、「薬剤」に敏感に反応します。まずは以下の表を参考に、原因を切り分けていきましょう。

| 原因カテゴリ | 主な特徴 | 薬剤師が問診で確認すべきこと |
| :--- | :--- | :--- |
| <strong>アルコール</strong> | 飲酒量・期間に比例して上昇 | 毎日の飲酒量、休肝日の有無、飲酒歴 |
| <strong>薬剤性</strong> | 服用開始数週間～数ヶ月で発症 | 新規導入薬、サプリメント、健康食品の併用 |
| <strong>非アルコール性</strong> | 肥満や代謝疾患（脂肪肝）がベース | 食生活、BMIの変化、糖尿病・脂質異常症の有無 |

### 🔍 「薬だけ」ではない理由
薬による肝障害（薬剤性肝障害）の多くは、γ-GTP単独上昇よりも、AST/ALTの有意な上昇を伴うことが多いです。γ-GTPのみが突出して高い場合は、アルコールや脂肪肝、あるいは胆石症などの器質的疾患を優先的に疑う必要があります。

---

## 💊 2. 薬剤性肝障害を疑う際の「チェックリスト」

もし薬の副作用を疑う場合は、以下の項目を確認しましょう。

<div style="background-color: #f0fdf4; padding: 1.5rem; border-radius: 8px; border: 1px solid #bbf7d0;">
  <strong>✅ 薬剤性肝障害を疑うポイント</strong>
  <ul style="margin-top: 0.5rem; color: #166534;">
    <li>新しく飲み始めた薬があるか（特に開始1〜3ヶ月以内）</li>
    <li>健康食品、プロテイン、ダイエットサプリを飲んでいないか</li>
    <li>他科受診を含め、服用薬の数が増えていないか</li>
    <li>皮膚の痒みや黄疸、全身倦怠感などの自覚症状はあるか</li>
  </ul>
</div>

特に<strong>「サプリメント」</strong>は見落としがちです。ウコンや肝臓エキスなどの「肝臓に良さそうなもの」が、逆に代謝負担をかけているケースも珍しくありません。

---

## 📊 3. γ-GTPを上昇させやすいお薬の特徴

もちろん、薬が原因でγ-GTPが上がることがあります。肝臓の酵素誘導（肝薬物代謝酵素が活性化すること）によって数値が上昇する薬剤は以下の通りです。

| 薬剤の種類 | 代表例 | 備考 |
| :--- | :--- | :--- |
| <strong>抗てんかん薬</strong> | フェノバルビタール、フェニトイン | 酵素誘導が強く、γ-GTP上昇は頻発する |
| <strong>抗生物質</strong> | 一部のセフェム系、マクロライド系 | 投与期間中の一過性の上昇が多い |
| <strong>抗真菌薬</strong> | イトラコナゾールなど | 代謝負担が大きく、定期的なモニタリングが必要 |
| <strong>脂質異常症薬</strong> | スタチン系（ロスバスタチン等） | 稀だが、肝機能障害として発現することがある |

---

## 🎯 4. 薬剤師ができる「実践的なアクション」

検査値だけで「薬の副作用」と決めつけず、以下のようなアプローチをとることで、患者さんの安心感と医師との連携がスムーズになります。

1.  <strong>時系列の確認</strong>: 検査値の推移と、薬の処方履歴を照らし合わせる。
2.  <strong>問診による周辺状況の把握</strong>: 「最近、お酒の量は増えていませんか？」「何か新しいサプリを始めましたか？」と丁寧に聴取する。
3.  <strong>医師へのフィードバック</strong>: 「AST/ALTは安定していますが、γ-GTPのみ上昇しています。アルコール摂取の増加の可能性を確認しました」といった、<strong>背景情報を加えた報告</strong>を行う。

## 🌿 5. まとめ：広視野なモニタリングを

「γ-GTP＝薬の副作用」という固定観念は、時として患者さんから必要な治療薬を遠ざけてしまうリスクがあります。
一方で、薬剤師は患者さんの生活背景を最もよく知る立場です。「薬以外の要因」を冷静に見極め、生活改善のアドバイスと薬物療法のモニタリングを両輪で行うことが、私たち専門家に求められる役割ではないでしょうか。

次回の服薬指導では、数値だけでなく、患者さんの「最近のライフスタイル」にもぜひ耳を傾けてみてくださいね！
