# 💊 頭痛が増えたら？薬剤師が受診を勧めるべきサインとは

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の実務エピソード】</strong><br>
  ある日、薬局のOTCカウンターに「ここ1ヶ月、頭痛薬の減りが異常に早い。週に4回は飲んでいる」と相談に来られた患者さんがいました。以前から「いつもの片頭痛」とおっしゃっていたため、私も深く考えずに常用量の鎮痛薬をお渡ししました。<br><br>
  しかし、数日後、その患者さんが「薬を飲んでも痛みが取れず、呂律が回りにくくなっている気がする」と再来店されたのです。すぐに脳神経外科への受診を強く勧め、結果として慢性硬膜下血腫が見つかりました。「もっと早く専門医に繋いでいれば…」という、あの時のヒヤリとした感覚は今も忘れません。
</div>

日常診療において、頭痛は最も頻繁に遭遇する主訴の一つです。しかし、それが「いつもの片頭痛」なのか、それとも「見逃してはならない危険な頭痛」なのか、その判断は薬剤師にとっても極めて重要です。今回は、私たちが窓口でチェックすべき<strong>「受診勧奨のサイン（レッドフラッグ）」</strong>について整理します。

---

## ⚠️ 1. これを見たら即受診！頭痛の「レッドフラッグ」

頭痛を訴える患者さんが来た際、以下の項目が一つでも当てはまる場合は、OTCでの対処を中止し、速やかな医療機関への受診を勧めるべきです。

| チェック項目 | 具体的な内容 |
| :--- | :--- |
| <strong>突然の発症</strong> | 「バットで殴られたような」突発的な激痛（くも膜下出血の疑い） |
| <strong>強度の増大</strong> | 徐々に痛みが強くなっている、あるいは頻度が増している |
| <strong>随伴症状</strong> | 麻痺、言語障害、視力障害、痙攣、意識障害を伴う |
| <strong>全身症状</strong> | 38度以上の発熱、首の硬直（項部硬直）を伴う |
| <strong>既往歴</strong> | がん、免疫不全状態、HIV感染などの基礎疾患がある |
| <strong>年齢層</strong> | 50歳以降に初めて経験する頭痛、または頭痛の性質の変化 |

<div style="background-color: #fff1f2; border: 1px solid #fecdd3; padding: 1rem; border-radius: 8px; color: #9f1239; margin: 1.5rem 0;">
  <strong>💡 薬剤師の判断のポイント</strong><br>
  「いつもの薬をください」という言葉を鵜呑みにせず、<strong>「以前と比べて痛みの強さや種類が変わっていないか？」</strong>という問いかけを必ず行いましょう。
</div>

---

## 💊 2. 薬剤の使用過多による頭痛（MOH）に注意

頭痛薬を連用する患者さんで特に注意が必要なのが、<strong>「薬剤の使用過多による頭痛（MOH: Medication-Overuse Headache）」</strong>です。

### 診断基準の目安
- 月に15日以上の頭痛がある
- 鎮痛薬（単純鎮痛薬）を月に15日以上使用している
- 3ヶ月を超えて連用が続いている

<div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe; margin-bottom: 1.5rem;">
  <strong>【薬剤師からのアドバイス】</strong><br>
  「薬を飲まないと痛くなる」という悪循環に陥っている患者さんには、単なる薬の提供ではなく、<strong>「痛み止めを飲む回数を減らすための予防療法」</strong>の必要性を説明し、専門医の受診を促すことが根本解決への近道です。
</div>

## 🌿 4. まとめ：薬剤師が「門番」になるために

頭痛はありふれた症状だからこそ、重大な疾患を見逃さない「ゲートキーパー」としての役割が薬剤師には求められています。

*   <strong>「いつもの」という言葉を疑う姿勢を持つ</strong>
*   <strong>レッドフラッグ（突然の激痛、麻痺など）は即専門医へ</strong>
*   <strong>薬の連用（MOH）には、予防療法の選択肢を提示する</strong>

私たち薬剤師が適切なタイミングで受診を促すことで、患者さんの将来的な健康リスクを大きく下げることができます。「薬を売って終わり」ではなく、「その先にある健康を見守る」という意識を、明日からのカウンター業務にぜひ活かしてみてください。
