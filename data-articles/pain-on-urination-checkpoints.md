# 排尿時に痛みが出たら？薬剤師が確認したいポイント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場のリアル：その「痛み」は膀胱炎？】</strong><br>
  ある日、顔色の悪い患者さんが来局され、「昨夜から排尿時にツンとした痛みがある。市販の膀胱炎薬を飲みたい」と相談されました。私は反射的に市販薬を案内しようとしましたが、念のため他の症状がないか確認したところ、「実は昨日から38度の熱が出ていて、腰のあたりも叩くと痛いんです」とのこと。<br><br>
  慌てて市販薬の提供を控え、すぐに医療機関への受診を強く勧奨しました。後日談を聞くと、診断は「腎盂腎炎」。あの時、単なる膀胱炎と判断して市販薬を販売していたらと考えると、ゾッとした経験があります。
</div>

「排尿時に痛みがある」という訴えは、調剤薬局やOTC販売において非常にポピュラーですが、同時に<strong>命に関わる疾患を見逃してはいけない重要なサイン</strong>でもあります。今回は、患者さんの訴えから薬剤師が読み取るべき「鑑別のヒント」を整理します。

---

## 🔍 1. 薬剤師が優先的に確認すべき「4つの質問」

患者さんが「排尿時の痛み」を訴えた場合、市販薬や軽症治療を検討する前に、必ず以下の項目を確認しましょう。

| 確認項目 | チェックの目的 |
| :--- | :--- |
| <strong>発熱の有無</strong> | 腎盂腎炎など、上部尿路感染症の疑い（要受診） |
| <strong>背部・側腹部叩打痛</strong> | 腎盂腎炎の可能性（コステロ徴候の確認） |
| <strong>症状の持続期間</strong> | 急性か慢性か、頻回か |
| <strong>性器周辺の違和感</strong> | 尿道炎や性感染症（STI）の可能性 |

### 🚨 受診勧奨の「レッドフラッグ」
以下の場合は、薬局での対応（市販薬販売や経過観察）を中止し、速やかに泌尿器科等の受診を勧めてください。
*   <strong>高熱（38度以上）を伴う</strong>
*   <strong>悪寒戦慄がある</strong>
*   <strong>背部痛や腰痛が強い</strong>
*   <strong>症状が数日以上改善しない、または悪化している</strong>
*   <strong>血尿が激しい</strong>

## ⚠️ 3. 膀胱炎治療における薬剤師の服薬指導ポイント

医療機関で膀胱炎と診断された患者さんに対し、薬剤師として伝えるべき重要事項です。

1.  <strong>「症状が消えても飲み切る」</strong>: 抗菌薬を自己判断で中断すると、耐性菌のリスクや再発の原因になります。
2.  <strong>「水分をしっかり摂る」</strong>: 尿量を増やすことで、細菌を洗い流す「排菌」を促すことが重要です。
3.  <strong>「我慢しない」</strong>: 尿意を感じたら我慢せず、こまめにトイレに行く習慣をアドバイスしましょう。
4.  <strong>「生活習慣の改善」</strong>: 性交渉後の排尿や、デリケートゾーンの清潔保持など、再発予防のアドバイスも有効です。

---

## 🌿 4. まとめ：薬剤師の役割は「適切なトリアージ」

排尿時の痛みは非常に不快な症状であり、患者さんは一刻も早い解決を求めています。しかし、その裏には「単なる膀胱炎」ではない深刻な疾患が隠れている可能性があります。

*   <strong>「いつから？」「熱は？」「腰は痛くない？」</strong> という3つの質問を徹底する。
*   少しでも全身症状（発熱・腰痛）があれば、<strong>迷わず受診勧奨</strong>を行う。
*   自己判断での市販薬使用に注意を促す。

このトリアージ（優先順位判断）を徹底することで、患者さんの安全を守り、地域の医療機関との連携もより強固なものになります。日々の業務で、ぜひ活用してくださいね！
