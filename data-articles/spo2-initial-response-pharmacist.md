# 🚨 SpO₂が急に低下した患者への初期対応

<div style="background-color: #fef2f2; border-left: 4px solid #ef4444; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の在宅訪問中、独居の患者様宅を訪問すると、いつもの穏やかな表情とは違い、肩で息をされていることに気づきました。慌ててパルスオキシメーターを装着すると、数値は「84%」。<br><br>
  パニックになった私は、すぐに「救急車を呼ぶべきか？」と迷いながら、まずは手元にあった吸入薬を使いそうになりました。しかし、ふと立ち止まりました。「これは喘息の発作なのか、それとも心不全の増悪なのか？ 今ここで判断を誤れば、状況を悪化させてしまうかもしれない……」。結局、主治医へ即座に状況を伝達し、指示を仰ぐことで大事に至りませんでしたが、その時の心臓の鼓動は今でも忘れません。
</div>

呼吸器系の疾患を抱える患者様を多く担当する薬剤師にとって、SpO₂の急低下はまさに「現場の緊急事態」です。今回は、いざという時に慌てず冷静に判断するための初期対応のポイントを解説します。

---

## 🔍 1. 数値だけで判断しない！まず見るべき「身体サイン」

パルスオキシメーターの数値はあくまで一つの指標です。機械の誤差や末梢循環不全（冷感など）で低く出ている可能性も否定できません。まずは<strong>「患者様の全体像」</strong>を確認しましょう。

### 薬剤師が確認すべき「緊急度チェックシート」

| 確認項目 | 緊急度：高（即時通報） | 緊急度：中（経過観察・連絡） |
| :--- | :--- | :--- |
| <strong>意識</strong> | 呼びかけに反応なし、傾眠 | 普段と変わらない |
| <strong>呼吸数</strong> | 30回/分以上、努力呼吸 | 20〜25回/分程度 |
| <strong>SpO₂</strong> | 85%以下（持続） | 88〜90%で安定 |
| <strong>顔色</strong> | 口唇チアノーゼ、冷や汗 | 通常通り |

## 💊 2. 薬剤師ができる「病態推測」と初期対応

SpO₂が低下したとき、背景にどのような疾患が潜んでいる可能性があるかを考えるのが薬剤師の役割です。

### 代表的な原因と初期対応

1. <strong>気管支喘息・COPD急性増悪</strong>
   - <strong>確認:</strong> 喘鳴（ヒューヒュー、ゼーゼー）があるか。
   - <strong>対応:</strong> 主治医の指示に基づき、レスキュー薬（サルブタモール吸入など）の使用準備。
2. <strong>心不全の増悪</strong>
   - <strong>確認:</strong> 下肢の浮腫、頸静脈の怒張、起座呼吸。
   - <strong>対応:</strong> 患者様を楽な姿勢（背中を起こす）に。利尿薬の飲み忘れを確認。
3. <strong>誤嚥・肺炎</strong>
   - <strong>確認:</strong> 痰の音、熱、食事直後の咳き込み。
   - <strong>対応:</strong> 口腔内の異物除去、吸引の実施（可能な場合）。

## 📞 3. 医師への伝達方法（SBAR法）

焦って報告すると、必要な情報が漏れてしまいます。以下のフレームワークを意識して報告しましょう。

- <strong>Situation（状況）</strong>: 「〇〇様、現在SpO₂が85%まで低下しています。」
- <strong>Background（背景）</strong>: 「基礎疾患にCOPDがあり、今朝から痰が増えていました。」
- <strong>Assessment（評価）</strong>: 「聴診でラ音を認め、呼吸回数も28回と多呼吸です。」
- <strong>Recommendation（提案・指示）</strong>: 「救急搬送を検討すべきか、指示をいただけますでしょうか？」

---

## 🌿 4. 最後に：平時の備えが命を救う

急変時の対応は、日頃のモニタリングから始まります。訪問時に「いつもと何かが違う」という違和感を言葉にして記録に残しておくことが、結果的に患者様の命を救うことに繋がります。

薬剤師として、薬の管理だけでなく<strong>「呼吸状態のモニタリング」</strong>もプロフェッショナルとして実践していきましょう。数値の変化を見逃さない観察力が、私たち薬剤師の最も頼れる武器になるはずです。
