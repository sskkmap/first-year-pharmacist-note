# 💊 食後に胃が痛くなるようになったら？薬剤師が考えたいこと

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  薬局に、「最近、食後に胃が痛くなる」と来局された患者さんがいらっしゃいました。私は即座に「胃酸過多かな？」と判断し、市販の制酸薬をおすすめしようとしました。しかし、念のため詳しく聞くと、実は痛みがあるのは「食後すぐ」ではなく「食後30分〜1時間」経過してからで、さらには最近痩せてきているとのこと。<br><br>
  単なる胃もたれと思い込み、市販薬で様子を見させてしまっていたら、胃潰瘍や悪性腫瘍の発見を遅らせていたかもしれません。「食後の胃痛＝胃酸過多」という安易な決めつけが、どれほど危険かと思い知らされた経験でした。
</div>

患者さんからの「胃が痛い」という訴えは、日常業務で非常によく遭遇します。しかし、一口に「食後の胃痛」といっても、その原因は様々です。私たち薬剤師には、単に薬を渡すだけでなく、<strong>「受診を促すべきレッドフラッグ（危険な徴候）」を見抜く力</strong>が求められています。

---

## 🔍 1. まずは「レッドフラッグ」を確認しよう

食後の胃痛を訴える患者さんに対し、服薬指導の前に以下のポイントを必ず確認しましょう。これらに該当する場合は、薬局での対応ではなく、速やかに医師の診断を仰ぐことが重要です。

| チェック項目 | 疑われる疾患・状態 |
| :--- | :--- |
| <strong>体重減少</strong>（意図しないもの） | 悪性腫瘍、重度の消化吸収障害 |
| <strong>嚥下困難・通過障害</strong> | 食道がん、食道アカラシア |
| <strong>黒色便・吐血</strong> | 上部消化管出血（潰瘍、腫瘍） |
| <strong>貧血の兆候</strong>（ふらつき、顔色が悪い） | 慢性的な出血による貧血 |
| <strong>持続する激痛・夜間の痛み</strong> | 胃潰瘍、重度の逆流性食道炎 |

## 🎯 3. 機能性ディスペプシア（FD）という視点

特に「検査では異常がないのに、食後の胃痛や胃もたれが続く」という患者さんには、<strong>機能性ディスペプシア（FD）</strong>の可能性があります。

- <strong>ポイント</strong>: 精神的ストレスや不規則な生活が胃の動きを鈍らせていることが多いため、胃薬だけでなく、以下の生活指導をセットで行うことが薬剤師の腕の見せ所です。
    - <strong>食事</strong>: ゆっくりよく噛んで食べる（消化の助け）。
    - <strong>ストレス</strong>: 自律神経を整えるような休息の提案。
    - <strong>嗜好品</strong>: カフェインやアルコール、香辛料などの刺激物を控える。

## ⚠️ 4. 「なんとなく」の市販薬購入を止めるために

患者さんが「とりあえず市販の胃薬で…」と考えている時こそ、私たち薬剤師の介入が必要です。

<div style="background-color: #fef2f2; border: 1px solid #fca5a5; padding: 1rem; border-radius: 8px; color: #991b1b; margin-top: 1rem;">
  <strong>🚨 薬剤師からのアドバイス</strong><br>
  「2週間ほど市販薬を試しても改善しない場合は、胃の粘膜の状態を詳しく調べるためにも、一度内視鏡検査（胃カメラ）を受けてください」と明確な期限を伝えてください。
</div>

「薬が効いているから大丈夫」と過信して、重大な疾患の発見を遅らせることがないよう、<strong>「薬は対症療法である」</strong>という限界を患者さんと共有しましょう。

## 🌿 まとめ

食後の胃痛を訴える患者さんの背後には、ただの消化不良から消化器系の疾患まで、様々なリスクが隠れています。
1. <strong>問診でレッドフラッグがないか確認</strong>
2. <strong>症状に合わせて適切な薬剤を選択</strong>
3. <strong>改善がない場合の受診勧奨を怠らない</strong>

この3ステップを意識するだけで、薬剤師として患者さんに提供できる価値は大きく変わります。明日からの服薬指導で、ぜひ意識してみてくださいね！
