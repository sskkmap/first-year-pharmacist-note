# 水を飲む量が増えたら？糖尿病だけではない原因とは

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  先日、70代の患者様から「最近急に喉が渇くようになって、一日中お茶を飲んでいるの。糖尿病かしら？」と相談を受けました。検査結果を見るとHbA1cは正常。私は「糖尿病ではないようなので、加齢による口渇かもしれませんね」と軽く答えてしまいました。<br><br>
  しかし、後日その方は「口渇のせいで眠れない」と受診し、実は服用中だった利尿薬の影響と、さらに背後に隠れていた別のホルモン疾患が見つかったのです。「糖尿病ではない」というだけで安心せず、薬剤性や疾患の広がりを考慮すべきだったと、深く反省した出来事でした。
</div>

患者様から「最近水を飲む量が増えた（多飲）」という相談を受けた際、どうしても「糖尿病」を真っ先に疑いがちです。しかし、実はその背後には<strong>薬剤の影響や、内分泌系の疾患</strong>など、見逃してはいけない原因が潜んでいることがあります。

今回は、現場で役立つ「多飲・多渇」の鑑別ポイントについて解説します。

---

## 🔍 1. 多飲・多渇の原因となる疾患・要因

多飲を疑う場合、まずは「高血糖による浸透圧利尿」以外に目を向ける必要があります。主な原因を整理しました。

| カテゴリ | 主な原因・疾患 |
| :--- | :--- |
| <strong>代謝・内分泌</strong> | 糖尿病、尿崩症（中枢性・腎性）、甲状腺機能亢進症、高カルシウム血症 |
| <strong>薬剤性</strong> | 利尿薬、抗精神病薬、リチウム、抗コリン薬 |
| <strong>心因性</strong> | 心因性多飲（精神疾患に関連） |
| <strong>その他</strong> | 加齢、慢性腎不全、シェーグレン症候群 |

### 🧠 尿崩症を疑うサイン
糖尿病の検査が陰性であっても、<strong>「尿が異常に多い（多尿）」かつ「喉が異常に渇く」</strong>場合、尿崩症を鑑別する必要があります。
- <strong>中枢性尿崩症</strong>: バソプレシン（抗利尿ホルモン）の分泌低下。頭部外傷や腫瘍などが原因。
- <strong>腎性尿崩症</strong>: 腎臓がバソプレシンに反応しない状態。リチウム（躁うつ病治療薬）の長期服用が有名です。

### 🚨 薬剤師が確認すべきチェックリスト
患者様から「喉が渇く」と相談されたら、以下の質問をしてみましょう。

1. <strong>いつから始まったか？</strong>（新薬の追加時期と一致するか？）
2. <strong>多尿はあるか？</strong>（尿の回数や量が増えたか？）
3. <strong>喉の渇き以外に症状はないか？</strong>（体重減少、動悸、多汗、浮腫など）
4. <strong>水分摂取量は具体的にどれくらいか？</strong>（数リットルに及ぶ場合は要注意）

---

## 🩺 3. 糖尿病だけではない！「レッドフラッグ」を見逃さない

もし患者様が「糖尿病の検査は受けているが異常なし」と言った場合、以下の疾患を視野に入れ、必要であれば受診勧奨を行う勇気が必要です。

*   <strong>甲状腺機能亢進症（バセドウ病など）</strong>: 代謝が亢進するため、多飲・多尿とともに体重減少、動悸、手指の震えが見られます。
*   <strong>高カルシウム血症</strong>: 腎臓の濃縮能が低下し、口渇・多尿が生じます。高齢者では特に注意が必要です。
*   <strong>シェーグレン症候群</strong>: 唾液分泌低下による「口腔乾燥（ドライマウス）」がメインですが、間質性腎炎を合併して腎性尿崩症の症状を呈することもあります。
