# 🌡️ 体温が上がったら？感染症だけではない原因を考える

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場で感じた「違和感」のエピソード】</strong><br>
  ある病棟担当の日、術後感染を疑われ広域抗生剤が開始された患者さんがいました。しかし、投与開始から1週間経っても38度台の熱が続いています。血液検査ではCRPは下がっているのに、熱だけが引かない……。<br><br>
  「おかしいな」と思い、持参薬を含めた薬剤リストを見直すと、入院直前から開始された抗てんかん薬があることに気づきました。医師に「薬剤熱の可能性はありませんか？」と提案し、一時休薬したところ、翌日には嘘のように解熱。
  「熱＝抗生剤」という思考停止が、かえって患者さんの入院期間を延ばしてしまいかねないと痛感した出来事でした。
</div>

臨床現場で「発熱」は最も頻繁に遭遇するバイタルサインの変化ですが、その原因は多岐にわたります。私たち薬剤師が、薬学的知見から<strong>「非感染性の発熱」</strong>を見抜くためのポイントを整理していきましょう。

---

## 🔍 1. まずは「発熱（Fever）」と「高体温症（Hyperthermia）」を区別する

一言で「体温上昇」と言っても、医学的にはそのメカニズムによって2つに大別されます。この違いを理解することが、適切なアセスメントの第一歩です。

    <div style="font-weight: bold; color: #9f1239; margin-bottom: 0.5rem; border-bottom: 2px solid #fecdd3; padding-bottom: 0.25rem;">発熱 (Fever)</div>
    <div style="color: #be123c; font-size: 0.9em;">

      <br>例：感染症、腫瘍熱、膠原病など
    </div>

  <div style="background-color: #fffbeb; padding: 1rem; border-radius: 8px; border: 1px solid #fde68a;">
    <div style="font-weight: bold; color: #92400e; margin-bottom: 0.5rem; border-bottom: 2px solid #fde68a; padding-bottom: 0.25rem;">高体温症 (Hyperthermia)</div>
    <div style="color: #b45309; font-size: 0.9em;">

## 💊 2. 薬剤師が最も注意すべき「薬剤熱」

<strong>薬剤熱</strong>は、薬剤投与によって引き起こされる発熱です。全入院患者の数％に起こるとも言われ、決して珍しくありません。

### 薬剤熱の特徴
- <strong>投与開始からの期間</strong>: 7〜10日後が多い（数時間〜数週間と幅がある）。
- <strong>全身状態</strong>: 高熱のわりには本人は比較的元気（Relative bradycardia：比較的徐脈を伴うことも）。
- <strong>検査値</strong>: CRP上昇、好酸球増多が見られることがあるが、決定的なものはない。
- <strong>診断</strong>: 疑わしい薬の中止後、通常48〜72時間以内に解熱する。

### 原因となりやすい主な薬剤
| 薬剤分類 | 具体的な薬剤例 |
| :--- | :--- |
| <strong>抗菌薬</strong> | β-ラクタム系、サルファ剤、バンコマイシンなど |
| <strong>抗てんかん薬</strong> | フェニトイン、カルバマゼピン、ラモトリギン |
| <strong>抗不整脈薬</strong> | プロカインアミド、キニジン |
| <strong>降圧薬</strong> | メチルドパ、ヒドララジン |

## 🚨 3. 緊急対応が必要な「薬剤性」高体温症

発熱だけでなく、精神症状や筋硬直を伴う場合は、生命に危険が及ぶ重症副作用を疑う必要があります。

## 📈 4. その他の「非感染性」発熱

感染症以外でも、以下の疾患や状態では体温が上昇します。

### 🎗️ 腫瘍熱 (Tumor Fever)
がん細胞自体が産生するサイトカイン（IL-6など）によって起こります。
- <strong>特徴</strong>: 抗菌薬に反応せず、<strong>ナプロキセン（ナイキサンなど）によく反応する</strong>（Naproxen test）という特徴があります。

### 🦋 内分泌性（甲状腺クリーゼなど）
甲状腺ホルモンの過剰により代謝が異常亢進し、産熱が放熱を上回ります。
- <strong>薬剤師の視点</strong>: 抗不整脈薬のアミオダロン（アンカロン）による甲状腺機能異常や、チェックポイント阻害薬（免疫関連副作用：irAE）による内分泌障害も念頭に置く必要があります。

## 🎯 5. 薬剤師ができる「発熱アセスメント」のチェックリスト

多忙な業務の中でも、以下のステップで情報を整理すると、医師への提案がスムーズになります。

<div style="background-color: #f0fdf4; border: 1px solid #bbf7d0; padding: 1.5rem; border-radius: 8px; color: #166534;">
  <h3 style="margin-top: 0; color: #15803d;">✅ 薬剤師のアセスメント・ステップ</h3>
  <ul style="list-style-type: none; padding-left: 0;">
    <li style="margin-bottom: 0.75rem;"><strong>1. タイムラインの確認:</strong> 発熱の数日前〜数週間前に新規開始・増量された薬はないか？</li>
    <li style="margin-bottom: 0.75rem;"><strong>2. 随伴症状のチェック:</strong> 皮疹、好酸球増多（薬剤熱）、筋強剛（悪性症候群）、下痢（セロトニン症候群）はないか？</li>
    <li style="margin-bottom: 0.75rem;"><strong>3. 検査値の乖離を見る:</strong> CRPは下がっているのに発熱だけ続いている、あるいは白血球数は正常なのに高熱が出ていないか？</li>
    <li style="margin-bottom: 0.75rem;"><strong>4. 薬剤の特性を考える:</strong> 抗生剤の「中ダメ（投与中盤での発熱）」は薬剤熱の典型パターン。</li>
  </ul>
</div>

## 🌿 6. まとめ：多角的な視点が患者を救う

「熱が出たら解熱剤」という対応はあくまで対症療法です。特に、解熱剤（NSAIDsやアセトアミノフェン）が効きにくい発熱や、原因不明の遷延する発熱において、私たち薬剤師の<strong>「薬歴という名のタイムライン」</strong>を確認するスキルは大きな武器になります。

冒頭のエピソードのように、感染症の治療中に起こる薬剤熱は非常に見落とされやすいものです。
「この熱、本当にバイ菌のせいかな？」と一度立ち止まって考える。そんな、薬のプロフェッショナルとしての視点を、ぜひ明日からの業務に活かしてみてください。
