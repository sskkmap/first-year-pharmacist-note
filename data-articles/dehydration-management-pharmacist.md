# 脱水が疑われる患者への対応：薬剤師が見極めるべき重症度とケアのポイント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の午後、血圧の薬を受け取りに来られた高齢の患者さんが、「最近、立ちくらみがひどくて…薬の副作用かしら？」と相談されました。血圧は安定しており、服用状況も良好。しかし、よく見ると肌の張り（皮膚の膨隆）が弱く、口唇が乾燥していました。<br><br>
  「ただの夏バテだろう」と見過ごしてしまえば、そのまま脳梗塞や腎機能低下を招いていたかもしれません。脱水は単なる水分不足ではなく、<strong>持病の悪化や服薬のリスクを倍増させる因子</strong>であることを、この時痛感しました。

薬剤師は、患者さんの「顔色」や「わずかな変化」に一番早く気づける立場にいます。今回は、日々の調剤業務で役立つ「脱水の兆候」の見抜き方と、プロとしてのアドバイスの仕方を整理します。

特に<strong>高齢者</strong>は喉の渇きを感じにくく、自分では気づかないうちに脱水が進行します。また、<strong>降圧薬（特に利尿薬）やSGLT2阻害薬</strong>を服用中の方は、脱水による急性腎障害（AKI）のリスクが高いため、優先的に確認が必要です。

## 🥤 2. 経口補水液（ORS）とスポーツドリンクの違い

患者さんから「ポカリスエットを飲めばいいの？」と聞かれることがよくあります。薬剤師として、その違いを正しく説明しましょう。

| 特徴 | スポーツドリンク | 経口補水液 (ORS) |
| :--- | :--- | :--- |
| <strong>主な目的</strong> | 水分・糖分の補給 | 脱水状態の補正（水分・電解質の迅速な吸収） |
| <strong>糖分</strong> | 多め（吸収はやや遅い） | 適量（吸収を速める設計） |
| <strong>電解質濃度</strong> | 低め | 高め（塩分をしっかり補給） |
| <strong>適した場面</strong> | 日常の水分補給、軽い運動時 | <strong>発熱、嘔吐、激しい発汗時</strong> |

<div style="background-color: #fef2f2; border: 1px solid #fecaca; padding: 1rem; border-radius: 8px; color: #991b1b; margin-top: 1rem;">
  <strong>💡 薬剤師からのアドバイスポイント：</strong><br>
  「脱水気味だな」と感じる時は、糖分が多く吸収の遅いスポーツドリンクではなく、<strong>電解質バランスの整った経口補水液（OS-1など）を少量ずつ頻回に飲む</strong>よう推奨してください。
</div>

## 🚑 3. 「いつ病院へ行くべき？」緊急度の見極め

脱水が疑われる場合、すべての患者さんを薬局でケアできるわけではありません。以下の場合は、速やかな受診（救急要請も含む）を勧める必要があります。

1. <strong>意識がぼんやりしている・呼びかけへの反応が鈍い</strong>（意識障害）
2. <strong>水分を全く受け付けない（嘔吐がひどい）</strong>
3. <strong>血圧が著しく低い、あるいは脈が異常に速い</strong>
4. <strong>尿が半日以上出ていない</strong>

これらのサインがある場合は、「様子を見ましょう」と帰すのではなく、ご家族へ連絡をとる、あるいは近隣の医療機関へ繋ぐことが私たちの重要な役割です。

## 🌿 4. 薬剤師にできる「服薬指導」の工夫

脱水リスクのある患者さんへの指導では、薬そのものの見直しも大切です。

*   <strong>利尿薬・SGLT2阻害薬</strong>: 脱水時は一時的な休薬を検討するため、医師へ疑義照会を行うフローを確認しておく。
*   <strong>服用タイミング</strong>: 喉が渇く前に水分を摂る習慣づけ（枕元に水をおくなど）を提案する。
*   <strong>生活指導</strong>: お風呂上がりや起床時のコップ一杯の水分補給を習慣化してもらう。

### まとめ：先手必勝の「気づき」が救命につながる

脱水は「たかが水不足」と軽視されがちですが、特に高齢者や持病を持つ方にとっては命に関わる緊急事態の入り口です。「いつもと顔色が違うな」「薬の説明の反応が鈍いな」という、<strong>薬剤師の直感的な気づき</strong>が患者さんのQOLを守ります。
