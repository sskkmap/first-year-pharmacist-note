# 排尿回数が減ったら？腎機能悪化を疑うべき？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ノート：その「尿減少」はどこから？】</strong><br>
  先日、70代の患者さんから「最近、薬局で測ってもらう血圧は安定しているのに、おしっこの回数が減った気がする」と相談を受けました。その方は降圧薬に加え、最近新たにNSAIDsを頓服で飲み始めたばかりでした。<br><br>
  私は反射的に「腎機能の低下？」と焦りましたが、詳細を伺うと、夏場で水分摂取が減っていたことによる「生理的な濃縮」の可能性も浮上。薬の副作用と加齢による腎機能変化、そして生活背景の判断が難しいケースでした。安易に「薬のせいですね」とも「気のせいですよ」とも言えない、臨床現場ならではの難しさを痛感した出来事です。
</div>

日常業務で患者さんから「尿の回数が減った」と聞くと、腎機能への影響を懸念される薬剤師の方は多いはずです。しかし、排尿減少は必ずしも腎機能の急性的な悪化だけを意味するわけではありません。今回は、薬剤師が現場で判断するために押さえておくべき視点を整理します。

---

## 🔍 1. まずは「尿量」と「回数」を切り分ける

患者さんの「排尿回数が減った」という訴えには、大きく分けて3つのパターンがあります。

| パターン | 主な原因の推測 | 薬剤師が確認すべきこと |
| :--- | :--- | :--- |
| <strong>尿量自体が減少</strong> | 脱水、腎血流量低下、心不全 | 水分摂取量、体重変化、浮腫の有無 |
| <strong>残尿感・頻尿</strong> | 前立腺肥大症、尿路感染症 | 排尿時の痛み、残尿感、勢いの低下 |
| <strong>夜間尿が減少</strong> | 睡眠の質改善、利尿薬の服用タイミング | 服用している利尿薬の用法、生活リズム |

「回数が減った」という訴えに対し、<strong>「おしっこの量は以前と比べてどうですか？（コップ1杯分程度でしょうか、それとも数滴でしょうか）」</strong>と、具体的な量を聞き出すことが重要です。

---

## ⚠️ 2. 薬剤性腎障害を疑うべき「警告サイン」

腎機能が悪化して尿量が減っている場合、それは薬剤による急性腎障害（AKI）の初期兆候かもしれません。以下のチェックリストを参考に、該当する患者さんがいないか確認してください。

### 【薬剤師がチェックすべき危険因子】
1.  <strong>腎毒性薬剤の重複使用</strong>: NSAIDs、抗菌薬（アミノグリコシド系など）、造影剤使用後など。
2.  <strong>RA系阻害薬の開始・増量</strong>: ACE阻害薬やARBを開始・増量直後は腎血流量が一時的に低下しやすいため、注意が必要です。
3.  <strong>利尿薬の過剰投与</strong>: 脱水を助長し、腎前性腎不全を招くリスクがあります。

<div style="background-color: #fff1f2; border: 1px solid #fecdd3; padding: 1rem; border-radius: 8px; color: #9f1239;">
  <strong>🚨 緊急度高！即時相談を促すべきサイン</strong><br>
  ・急激な体重増加（浮腫の出現）<br>
  ・血尿や尿の色の変化（濃い褐色など）<br>
  ・強い倦怠感や吐き気、食欲不振<br>
  ・下肢の明らかなむくみ
</div>

## 💡 3. 実務で使える！患者さんへの服薬指導ポイント

患者さんとの会話の中で、「腎機能のチェック」を自然に行うためのテクニックを紹介します。

### 「腎機能」という言葉を使わずに確認する
「腎臓の数値はどうですか？」と聞いても、患者さんは「よくわからない」と答えることがほとんどです。次のような質問に置き換えてみてください。

*   <strong>「最近、足のむくみを感じることはありますか？」</strong>
    *   腎機能低下による水分排泄不良を確認。
*   <strong>「お水や飲み物をとる量は、以前と変わっていませんか？」</strong>
    *   脱水による尿量減少か、病的なものかを鑑別。
*   <strong>「お薬を飲んでから、立ちくらみやふらつきを感じることはありませんか？」</strong>
    *   血圧低下に伴う腎血流量低下の可能性を確認。

---

## 📊 4. 腎機能変化に伴う薬剤調整のフロー

もし患者さんに腎機能低下の疑いがある場合、私たちは医師に対して「どのような疑義照会・提案ができるか」をシミュレーションしておく必要があります。

<div style="overflow-x: auto;">

| 腎機能低下時の対応 | 具体的なアクション |
| :--- | :--- |
| <strong>用量調節</strong> | eGFRを確認し、腎排泄型薬剤（メトホルミン等）の用量を見直す |
| <strong>休薬検討</strong> | NSAIDsなどの腎負担が大きい薬剤の頓服を控えるよう提案 |
| <strong>モニタリング</strong> | 血液検査（Cre, eGFR, BUN）の定期的な確認を医師へ依頼 |
| <strong>水分補給</strong> | 脱水リスクがある場合、適切な水分摂取を指導しつつ指導箋を活用 |

</div>

## 🌿 5. まとめ：薬剤師ができる「腎の門番」

「排尿回数が減った」という訴えは、患者さん自身が感じる最初の小さな違和感です。これを単なる老化や体調不良で片付けず、<strong>「直近で変更された薬はないか？」「脱水はないか？」「浮腫は出ていないか？」</strong>と多角的に評価するのが薬剤師の腕の見せ所です。

特に高齢者の場合、多剤併用（ポリファーマシー）によって「良かれと思って飲んでいた薬」が、結果的に腎機能を圧迫しているケースも少なくありません。

もし「排尿回数が減った」という相談を受けた際は、焦らずまずは「量」と「むくみ」を確認し、必要であれば検査データの参照や医師への疑義照会を通じて、安全な薬物療法をサポートしていきましょう。
