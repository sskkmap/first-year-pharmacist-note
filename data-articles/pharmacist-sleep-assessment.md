# 💤 睡眠から読み解く薬剤師のアセスメント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  かつて私は、睡眠薬を漫然と服用し続けている患者さんから「最近、薬が効かなくて」と相談された際、脊髄反射的に「医師に相談して増量してもらいましょうか」と提案してしまいました。<br><br>
  しかし、よくよく話を聞いてみると、実は最近始めた別の疾患の治療薬（ステロイド剤）が夜間の覚醒を誘発していたのです。睡眠薬を増やす前に、服薬タイミングの指導や薬剤調整が必要だったケースでした。「眠れない」という主訴を単なる睡眠薬の有効性だけで判断してしまった、私のアセスメント不足が招いた苦い経験です。
</div>

患者さんの「眠れない」という言葉は、私たち薬剤師にとって<strong>「全身状態や他の薬剤の影響を精査せよ」</strong>というサインです。今回は、睡眠から患者の状態を読み解くアセスメントのポイントを解説します。

---

## 🔍 1. 不眠の「タイプ」で読み解く病態

睡眠薬を単に「眠らせるもの」として捉えるのではなく、不眠のタイプから背景因子を探ることが大切です。

| タイプ | 特徴 | 疑われる背景因子 |
| :--- | :--- | :--- |
| <strong>入眠障害</strong> | 寝つきが悪い | 不安、緊張、カフェイン摂取、概日リズムのズレ |
| <strong>中途覚醒</strong> | 夜中に何度も目が覚める | 加齢、睡眠時無呼吸症候群、抑うつ、排尿障害 |
| <strong>早朝覚醒</strong> | 朝早く目が覚めてしまう | うつ病（特に高齢者）、アルコール、加齢 |
| <strong>熟眠障害</strong> | 眠りが浅い | 睡眠時無呼吸症候群、薬剤の副作用、ストレス |

## 🛠 3. 薬剤師にできる「睡眠衛生」の指導（Sleep Hygiene）

薬の調整だけでなく、非薬物療法としての「睡眠衛生」を指導することも薬剤師の重要な役割です。以下のリストを活用し、患者さんとチェックしてみてください。

1.  <strong>カフェイン・アルコールの確認</strong>: 午後以降の摂取を控えるよう提案。
2.  <strong>昼寝の調整</strong>: 午後の短い仮眠（20分程度）はOKだが、夕方の長時間睡眠はNG。
3.  <strong>光のコントロール</strong>: 朝起きたら日光を浴び、夜はブルーライトを避ける。
4.  <strong>就床タイミング</strong>: 「眠くなってから布団に入る」。布団を「眠れない場所」にしない。

## 💡 まとめ：薬剤師のアセスメントフロー

最後に、患者さんから「眠れない」と言われた時に実行してほしいフローをまとめました。

1.  <strong>「いつから？」「どんなふうに？」を深掘りする</strong>（入眠障害か、中途覚醒か）。
2.  <strong>併用薬の副作用を疑う</strong>（特にステロイド、利尿薬、カフェインを含むOTC薬）。
3.  <strong>生活習慣の乱れをチェックする</strong>（昼寝、運動不足、ストレス）。
4.  <strong>医師への疑義照会を検討する</strong>（薬剤の影響が疑われる場合や、睡眠の質が著しく低い場合）。

睡眠薬を単に「出すだけ」「確認するだけ」ではなく、患者さんの生活と薬を俯瞰して読み解く。これこそが、私たち薬剤師に求められる「睡眠アセスメント」の真髄です。次回の服薬指導から、ぜひ取り入れてみてください。
