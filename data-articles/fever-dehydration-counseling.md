# 発熱と脱水を見逃さないための服薬指導

<div style="background-color: #fff7ed; border-left: 4px solid #fb923c; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  数年前の夏、高熱で解熱剤を求めて来局された患者さんがいました。事務さんが応対し、私はお薬の準備をしてお渡ししたのですが、その際、特に体調の詳細を深く聞き取らずに「お大事になさってください」と一言添えただけでした。<br><br>
  翌日、同じ患者さんが救急搬送されたと家族の方から連絡がありました。「熱があって食欲も水分も取れなかったのに、解熱剤を飲んで無理に動いてしまった」とのこと。もしあの時、私が「今の水分摂取量は？」「尿の色は？」と一言確認できていたら……。ただ解熱剤を渡すだけの「モノ売り」になっていた自分を猛省した出来事でした。
</div>

発熱時の服薬指導において、薬剤師が最も優先すべきは「解熱剤の渡し方」ではなく<strong>「脱水状態の評価」</strong>です。今回は、現場で明日から使える「脱水を見逃さないためのチェックポイント」を整理しました。

## 🔍 1. 脱水を見極める！薬剤師が聞くべき3つの質問

患者さんの状態を迅速に把握するために、以下の3項目を必ず確認しましょう。

| チェック項目 | 確認の意図 | 具体的な聞き方 |
| :--- | :--- | :--- |
| <strong>水分摂取状況</strong> | 飲水量の減少を確認 | 「今日、コップ何杯くらい水分が摂れていますか？」 |
| <strong>尿の状態</strong> | 腎血流量の低下を推測 | 「今日、おしっこは出ていますか？色は濃くないですか？」 |
| <strong>自覚症状</strong> | 循環血液量減少のサイン | 「立ち上がるときにフラフラしたり、頭が痛かったりしませんか？」 |

<div style="background-color: #f1f5f9; padding: 1rem; border-radius: 8px; margin: 1rem 0;">
  <strong>💡 豆知識：尿の色による判断</strong><br>
  尿の色が「濃い黄色〜茶褐色」で、かつ回数が極端に減っている場合、すでに中等度以上の脱水が疑われます。この場合は、お薬の服用以前に「医療機関への再受診（点滴治療の検討）」を促す判断が必要です。
</div>

## 💧 2. 経口補水液（OS-1等）の上手な推奨方法

「水やお茶」だけでは、発熱による発汗で失われた電解質を補うことができません。服薬指導時には、経口補水液の必要性を具体的に伝えましょう。

*   <strong>なぜ水ではダメなのか？</strong>
    *   水だけだと血液の浸透圧が下がり、さらなる利尿を促して脱水を悪化させることがあります。
*   <strong>「水」と「経口補水液」の使い分け</strong>

<div style="overflow-x: auto;">
| 特徴 | 水・お茶 | 経口補水液（OS-1等） |
| :--- | :--- | :--- |
| <strong>主目的</strong> | 日常の水分補給 | 欠乏した水分・電解質の迅速な補給 |
| <strong>推奨シーン</strong> | 平熱時、軽度の発汗 | 高熱、嘔吐、下痢、脱水気味の時 |
| <strong>薬剤師のアドバイス</strong> | 「水分は必要ですが、熱があるときは電解質も一緒に補う必要があります」 | 「ちびちびと回数を分けて飲むのが、吸収効率を高めるコツです」 |
</div>

## ⚠️ 3. 解熱鎮痛剤の使用に潜むリスク

解熱鎮痛剤（NSAIDs等）は、熱を下げて楽にしてくれる反面、脱水時には注意が必要です。

1.  <strong>NSAIDsと腎機能</strong>: 腎血流量が低下している脱水状態でNSAIDsを使用すると、急性腎障害のリスクが高まります。
2.  <strong>無理な活動の防止</strong>: 熱が下がると「治った」と勘違いし、動いてしまう患者さんが多いです。<strong>「熱が下がっても体内の水分バランスはすぐには戻らない」</strong>ことを強調してください。

<div style="background-color: #fef2f2; border: 1px solid #fecaca; padding: 1rem; border-radius: 8px; color: #991b1b; margin-top: 1rem;">
  <strong>🚨 薬剤師の判断基準（受診勧奨のサイン）</strong><br>
  以下の症状がある場合は、薬を渡す前に必ず医師の診察を優先させてください。<br>
  ・呼びかけに対する反応が鈍い（意識障害）<br>
  ・皮膚をつまんで戻すと、すぐに戻らない（皮膚緊張度の低下）<br>
  ・嘔吐が続いていて、経口摂取が全くできない
</div>

## 🌿 まとめ：薬を渡す前の「プラスアルファ」が命を守る

発熱の服薬指導は、<strong>「薬＋αの生活指導」</strong>がセットです。

*   <strong>「飲める？出てる？フラつく？」</strong>をまず聞く。
*   水分補給は<strong>電解質を含むもの</strong>を勧める。
*   <strong>解熱後も安静が必要</strong>であることを伝える。
