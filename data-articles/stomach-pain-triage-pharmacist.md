# 「胃が痛い」と言われたら？明日から使える胃痛トリアージの極意

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日、「ここ数日、胃がキリキリ痛むので市販の胃薬が欲しい」とご高齢の患者さんが相談に来られました。私は何気なく「よくある胃酸過多でしょう」と考え、制酸剤入りの胃薬を勧めて販売しました。<br><br>
  しかし翌週、その患者さんが「胃薬を飲んでも痛みが取れず、病院に行ったら胃潰瘍だった」と来店されました。幸い大事には至りませんでしたが、私は「単なる胃痛」と決めつけて市販薬で様子見をさせてしまったことを深く反省しました。胃痛の裏には、放置してはいけない病気が隠れている可能性があることを、この時痛感したのです。
</div>

「胃が痛い」という相談は、門前薬局やOTC販売の現場で最も多い相談の一つです。しかし、そこには<strong>「ただの胃酸過多」から「放置できない疾患」まで幅広い可能性</strong>が含まれています。

今回は、薬剤師が現場で自信を持って判断できるよう、胃痛のトリアージと、症状・薬効別の選び方についてまとめました。

---

## 🔍 1. 薬剤師が確認すべき「レッドフラッグ（危険信号）」

胃痛を訴える患者さんに対し、まずは市販薬で対応して良いか、医師の受診を優先すべきかを見極める必要があります。以下の項目に当てはまる場合は、<strong>「まずは受診を」と強く勧める</strong>ことが重要です。

| チェック項目 | 注意すべき疾患の可能性 |
| :--- | :--- |
| <strong>体重減少がある</strong> | 胃がんなどの悪性腫瘍 |
| <strong>黒色便が出ている</strong> | 上部消化管出血 |
| <strong>背中まで突き抜ける痛み</strong> | 膵炎、胆石症、大動脈解離 |
| <strong>嚥下困難（飲み込みにくい）</strong> | 食道がん、食道アカラシア |
| <strong>吐血・貧血症状がある</strong> | 重度の胃潰瘍、出血 |

これらを「スクリーニング」してから、お薬の提案へ進みましょう。

---

胃の不快感には「酸が過剰なタイプ」と「胃の動きが鈍いタイプ」があります。これらを混同して選ぶと、症状が改善しません。

## ⚠️ 3. 注意すべき薬剤師の「禁忌」と「併用薬」

調剤現場では、<strong>「胃薬と併用してはいけない薬」</strong>の確認も重要です。

*   <strong>H2ブロッカーと腎機能</strong>: 腎機能が低下している患者さんには用量調節が必要です。
*   <strong>PPI（プロトンポンプ阻害薬）の長期服用</strong>: 骨粗鬆症リスクや、ビタミンB12吸収障害などの長期副作用も考慮し、適宜医師への服薬期間確認（疑義照会）を検討しましょう。
*   <strong>抗血小板薬（バイアスピリン等）</strong>: NSAIDsや抗血小板薬を服用中の患者さんは、胃粘膜保護が必須です。「胃薬も一緒に出しておきますね」の一言が、重大な出血性潰瘍を防ぐことにつながります。

## 🎯 まとめ：薬剤師ができる「ワンランク上の提案」

胃痛の相談を受けたとき、単に「胃薬を出す」のではなく、以下のステップを意識してみてください。

1.  <strong>問診（レッドフラッグの確認）</strong>: 「最近、体重は減っていませんか？」「便の色は大丈夫ですか？」
2.  <strong>症状の分類</strong>: 「食後ですか？空腹時ですか？」を確認し、酸を抑えるべきか、動きを促すべきか判断する。
3.  <strong>生活指導のセット</strong>: 「ストレスやカフェイン、寝る前の食事なども胃痛の原因になりますよ」と、薬以外のアドバイスを添える。

「〇〇が痛い」という相談は、患者さんとの信頼関係を築くチャンスです。しっかりとした知識に基づいたトリアージで、患者さんの「安心」を支えていきましょう！
