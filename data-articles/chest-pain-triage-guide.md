# 胸が痛いと言われたら？受診勧奨を考えるべき危険なサインとは

<div style="background-color: #fff1f2; border-left: 4px solid #f43f5e; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  薬局の待合室で、「最近、胸のあたりが締め付けられるように痛いことがあって…」と高齢の患者様から相談を受けたことがあります。当時は「加齢に伴う逆流性食道炎かな？」と思い、市販の胃薬を勧めかけました。<br><br>
  しかし、直後に「冷や汗が出ることもある」という一言を聞いてハッとしました。急いでバイタルを確認すると脈が非常に速く、すぐに救急外来への受診を強く勧めました。結果として、その方は急性心筋梗塞で緊急カテーテル手術となり、一命を取り留めました。もし、あの時「胃薬」を出して帰宅させていたらと思うと、今でも背筋が凍る思いです。
</div>

薬剤師として働いていると、体調不良の相談を受けることは日常的です。しかし「胸の痛み」は、<strong>「ただの消化不良」から「命に関わる心疾患」まで幅が広く、判断を誤れば致命的</strong>になり得ます。

今回は、現場で「胸が痛い」と言われた際、私たちが最低限押さえておくべき危険なサインと判断基準を整理します。

---

## 🚨 1. 見逃してはいけない「レッドフラッグ」
胸の痛みを訴えられたとき、まずは「心臓のトラブル」を疑うことが鉄則です。以下の項目に一つでも当てはまる場合は、<strong>「様子を見ましょう」という選択肢は捨て、迷わず受診勧奨（あるいは救急搬送）</strong>を行います。

| 項目 | 具体的な症状・サイン |
| :--- | :--- |
| <strong>痛みの性質</strong> | 締め付けられる、圧迫感、焼けつくような痛み |
| <strong>持続時間</strong> | 15分以上続く、または安静にしていても治まらない |
| <strong>随伴症状</strong> | <strong>冷や汗、呼吸困難</strong>、吐き気、顔面蒼白 |
| <strong>放散痛</strong> | 左肩、顎（あご）、背中、左腕への痛みやしびれ |
| <strong>既往歴</strong> | 高血圧、糖尿病、脂質異常症、喫煙習慣がある場合 |

特に、<strong>「冷や汗」を伴う胸痛</strong>は、急性心筋梗塞などの致死的な心疾患において非常に特異度が高いサインです。

---

## 🩺 2. なぜ「薬剤師」が判断しなければならないのか？

患者様は「わざわざ病院に行くほどでもないかな」と自己判断しがちです。しかし、心筋梗塞などは<strong>「発症から治療までの時間（ゴールデンアワー）」が予後を大きく左右</strong>します。

私たち薬剤師の役割は、「診断」ではなく<strong>「緊急性のスクリーニング」</strong>です。

### 薬剤師が行うべきクイック・トリアージ
1. <strong>痛みの場所と質を確認</strong>: 「指一本でさせる痛み（筋肉痛の可能性）」か「全体的な圧迫感（心疾患の可能性）」か。
2. <strong>緊急性の高い随伴症状の確認</strong>: 冷や汗が出ていないか、息苦しさはあるか。
3. <strong>現在の服薬状況</strong>: 狭心症の既往があり、ニトログリセリンを処方されているか（処方があればすぐに使用を促す）。

## 🎯 4. 現場で使える「伝え方のコツ」

患者様を不安にさせすぎず、かつ確実に受診へ繋げるためのフレーズです。

*   <strong>「今は落ち着いているかもしれませんが、今の症状は心臓からのサインである可能性も否定できません。念のため、早急に循環器内科を受診してください。」</strong>
*   <strong>「もし、この後すぐにまた痛みが出たり、冷や汗が出てきたりした場合は、ためらわずに119番通報してください。」</strong>

「大丈夫ですよ」という言葉は、私たちにとって最もリスクの高い言葉です。「大丈夫」という確信が持てない限り、絶対にその言葉を使ってはいけません。

---

## 🌿 まとめ：安全側に倒す勇気を持つ

胸の痛みを訴える患者様に対して、最も怖いのは<strong>「見逃し」</strong>です。
逆流性食道炎や肋間神経痛であっても、それは医師が診断を下すべきこと。薬剤師は、「心臓に悪いものかもしれない」という最悪のケースを常に想定し、リスクを感じたらすぐに専門医へ繋ぐ勇気を持つことが、結果として患者様を守ることになります。

「ただの相談」が「救命」に繋がることもあります。日々の業務において、この緊張感と判断基準を常に忘れないようにしたいですね。
