# BNPが上がったら？心不全悪化をどう評価する？

<div style="background-color: #f0f9ff; border-left: 4px solid #0284c7; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  ある日、外来の処方箋でBNPの値が「前回500→今回1200」と大幅に上昇している患者さんを担当しました。処方内容は変更なし。つい「数値が倍以上になっているから、もっと利尿薬を増やした方がいいのでは？」と不安になり、処方医に「BNPが上がっていますが大丈夫でしょうか？」と電話をかけてしまいました。<br><br>
  すると先生から「昨日から風邪気味で水分摂取が減っていたんだよ。BNPは上がったけど、身体所見は落ち着いているから様子見だ」と返答が。数値の裏側にある「背景」を評価せず、数字だけを見て焦ってしまった自分を恥じた経験があります。
</div>

循環器内科以外の門前薬局や、在宅医療に関わる薬剤師にとって、検査値（BNP/NT-proBNP）の変化は判断に迷うポイントの一つですよね。今回は「BNPの数値だけにとらわれない、心不全評価の実践的な考え方」を深掘りします。

---

## 🔍 1. BNP上昇はすべて「心不全悪化」なのか？

BNP（脳性ナトリウム利尿ペプチド）は心負荷の重要な指標ですが、<strong>BNP＝心不全の悪化とは限りません。</strong>

<div style="display: grid; grid-template-columns: repeat(2, 1fr); gap: 1rem; margin: 1.5rem 0;">
  <div style="background-color: #f8fafc; padding: 1rem; border-radius: 8px; border: 1px solid #cbd5e1;">
    <div style="font-weight: bold; color: #334155; margin-bottom: 0.5rem;">BNPが上がる原因</div>
    <ul style="font-size: 0.9em; padding-left: 1.2rem;">
      <li>心機能の低下（心不全）</li>
      <li>心房細動などの不整脈</li>
      <li>腎機能低下（排泄遅延）</li>
      <li>加齢・高齢者</li>
    </ul>
  </div>
  <div style="background-color: #fff1f2; padding: 1rem; border-radius: 8px; border: 1px solid #fda4af;">
    <div style="font-weight: bold; color: #9f1239; margin-bottom: 0.5rem;">注意すべき「偽高値」要因</div>
    <ul style="font-size: 0.9em; padding-left: 1.2rem;">
      <li><strong>腎機能低下:</strong> eGFR低下により排泄されず上昇</li>
      <li><strong>薬剤の影響:</strong> ARNI（エンレスト）服用中は上昇する</li>
    </ul>
  </div>
</div>

特に注意が必要なのが、<strong>ARNI（サクビトリルバルサルタン）服用中</strong>です。ARNIはBNPの分解を抑制するため、服用開始後にBNP値が上昇します。これは「心不全が悪化した」のではなく、<strong>「薬の作用でBNPの分解が抑えられているだけ」</strong>という可能性が高いです。

---

## 🩺 2. 薬剤師が確認すべき「心不全のフィジカルサイン」

数値が上がったとき、薬剤師が窓口や在宅で確認できる項目をリスト化しました。これらを評価することで、医師への情報の質が変わります。

| 確認項目 | チェックのポイント |
| :--- | :--- |
| <strong>体重変化</strong> | 1週間で1〜2kg以上の急激な増加はないか？ |
| <strong>下肢の浮腫</strong> | 脛骨前縁を押して、あとが残る（圧痕性浮腫）か？ |
| <strong>呼吸状態</strong> | 息切れ、夜間に枕を高くしないと寝苦しい（起坐呼吸）か？ |
| <strong>自覚症状</strong> | 以前より活動量が減った、食欲不振があるか？ |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px;">
  <strong>💡 アドバイス:</strong> 「最近、靴がきつくないですか？」「夜、横になると苦しくなることはありますか？」といった具体的な問いかけが、早期発見のカギになります。
</div>

## 💊 3. 処方提案につなげるための思考プロセス

BNPが上がっていた場合、単に「心不全が悪化しました」と伝えるのではなく、以下のステップで情報を整理しましょう。

1. <strong>背景因子の確認</strong>: 感染症、脱水（逆に血液濃縮で値が上がることがある）、塩分摂取過多、服薬アドヒアランスの低下はないか。
2. <strong>腎機能のチェック</strong>: eGFRがベースラインより急激に低下していないか。
3. <strong>薬物療法の見直し</strong>: 利尿薬の調整は適切か？（過剰投与による脱水・低血圧リスクとの天秤）

### 心不全治療薬のポイント比較

<div style="overflow-x: auto;">
| 薬剤の種類 | 役割・注意点 |
| :--- | :--- |
| <strong>ループ利尿薬</strong> | 溢水除去の主役。低K血症、脱水に注意。 |
| <strong>ACEI/ARB/ARNI</strong> | 心負荷軽減。血圧低下、高K血症、腎機能悪化に注意。 |
| <strong>β遮断薬</strong> | 長期予後改善。心不全急性期には導入不可。 |
| <strong>SGLT2阻害薬</strong> | 心不全入院リスクを低減。脱水、尿路感染に注意。 |
</div>

## 🌿 4. まとめ：数値は「警告灯」、大事なのは患者さんの状態

BNPはあくまで「心臓からのSOSサイン」の一つに過ぎません。

*   <strong>数字が上がった＝必ず増薬が必要ではない</strong>
*   <strong>数字は平常＝心不全が完全に制御できているわけではない</strong>

薬剤師としては、検査値の推移という「客観データ」と、服薬指導時に見聞きする「患者さんの状態」を組み合わせて判断する能力が求められます。

もし、BNPが上昇しており、かつ浮腫や体重増加などの症状が見られる場合は、<strong>「BNPの上昇と、体重増加・浮腫の併発を確認しました。心不全の悪化の可能性を考慮して検討いただけますでしょうか」</strong>と医師に提案することで、臨床現場でより頼られる薬剤師になれるはずです。

次回の外来では、検査値を見る前に、まずは患者さんの表情や足元に注目してみてくださいね！
