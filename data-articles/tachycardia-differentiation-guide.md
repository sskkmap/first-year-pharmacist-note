# 心拍数が増えたら？副作用・疾患悪化・脱水の見分け方

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の服薬指導中、降圧薬を処方されている高齢の患者さんから「最近、安静にしていても心臓がドキドキするのよ」という相談を受けました。私は反射的に「降圧薬の影響で血圧が下がりすぎて、代償性に頻脈になっているのでは？」と疑い、血圧計をチェックしてもらったのですが、血圧は普段通りでした。<br><br>
  実はその後、受診した主治医の診断は「軽度の脱水による頻脈」でした。その日、暑さによる発汗と水分摂取不足が重なっていたのです。安易に「薬の副作用」と決めつけて指導していたら、根本的な脱水の発見が遅れていたかもしれない――そう気づいた時、ヒヤリとしました。
</div>

患者さんからの「動悸がする」「脈が速い」という訴えは、薬剤師にとって日常的ですが、実は背景にある原因を見極めるのが難しい主訴です。今回は、臨床現場で役立つ<strong>「心拍数増加（頻脈）の鑑別アプローチ」</strong>を整理します。

### 薬剤師が確認すべき3つの質問
1. <strong>「いつから、どんな時に速くなりますか？」</strong>（安静時か、動作時か、服薬後か）
2. <strong>「喉の渇き、尿の回数、便の状態はどうですか？」</strong>（脱水の有無）
3. <strong>「他に気になる症状はありますか？」</strong>（息切れ、浮腫、発熱、体重増加など）

---

## 💊 2. 頻脈を起こしやすい代表的な薬剤

「薬剤性」を疑う場合、処方内容をまず確認しましょう。

| 薬剤分類 | 頻脈の機序 | 注意すべきポイント |
| :--- | :--- | :--- |
| <strong>β刺激薬</strong> (気管支拡張薬) | 直接的な心筋刺激 | 吸入回数や過剰使用の確認 |
| <strong>血管拡張薬</strong> (ニトログリセリン等) | 反射性頻脈 (血圧低下の補償) | 起立性低血圧の有無を確認 |
| <strong>甲状腺ホルモン製剤</strong> | 代謝亢進による心拍数増加 | 用量調整が必要な可能性あり |
| <strong>抗不安薬・睡眠薬の離脱</strong> | 交感神経の過活動 | 急な中止がないか確認 |
| <strong>市販の風邪薬</strong> (カフェイン/プソイドエフェドリン) | 交感神経刺激 | OTCの重複使用がないか |

## 🚰 3. 「脱水」を見極めるためのヒント

特に高齢者では、脱水は無症状のまま進行し、心拍数増加という形で現れることが多いです。

*   <strong>循環血液量の減少</strong>: 体内の水分が減ると、心臓は少ない血液を全身に送るために、回数を増やして（心拍数を上げて）補おうとします。
*   <strong>薬剤師チェックポイント</strong>:
    *   <strong>舌の状態</strong>: 乾燥していないか？
    *   <strong>皮膚のツルゴール</strong>: 手の甲をつまんで、すぐに戻るか？
    *   <strong>尿の色</strong>: 濃くなっていないか？
    *   <strong>背景</strong>: 最近、下痢や嘔吐、夏場の発汗、あるいは節水意識などがないか。

## ⚠️ 4. 疾患悪化（心不全など）のサイン

心不全患者さんの場合、頻脈は<strong>「心負荷の増大」</strong>のサインであり、非常に危険です。

<div style="background-color: #fff7ed; border: 1px solid #fed7aa; padding: 1rem; border-radius: 8px;">
  <strong>緊急性が高い！すぐに受診を勧めるべきサイン</strong>
  <ul style="margin: 0.5rem 0 0 1.2rem;">
    <li>体重が短期間（数日）で1〜2kg以上急増している</li>
    <li>横になると息苦しい（起座呼吸）</li>
    <li>足のむくみが明らかにひどくなった</li>
    <li>脈が飛ぶ、リズムがバラバラである（心房細動の疑い）</li>
  </ul>
</div>

## 🎯 まとめ：薬剤師としてどう動くか

1. <strong>副作用を疑う</strong>: 直近で増量や新規追加された薬剤がないかを確認。
2. <strong>脱水を疑う</strong>: 日常的な水分摂取量や、脱水の身体徴候（口渇・尿量減少）を尋ねる。
3. <strong>心不全を疑う</strong>: 呼吸苦や浮腫を確認し、あれば速やかな受診を促す。

患者さんの「脈が速い」という訴えは、心臓からのSOSかもしれません。「薬のせいかな？」と断定せず、<strong>脱水や隠れた疾患の可能性を広げて聞き取る</strong>ことで、薬剤師としての臨床判断の質が大きく向上します。

もし、「いつものことだから」という返答であっても、患者さんの背景に変化がないか（季節、生活環境、食事量）を定期的にモニタリングする姿勢を大切にしていきましょう。
