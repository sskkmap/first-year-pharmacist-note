# 抗血栓薬服用患者の出血リスク評価

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  以前、心房細動でDOACを処方されている患者さんが来局された際のことです。患者さんは「膝が痛いから」と、長期間市販の鎮痛剤（NSAIDs）を併用されていました。「最近、歯茎からの出血が止まりにくい」という主訴を聞き、慌てて残薬を確認すると、出血リスクが極限まで高まっていたのです。<br><br>
  「薬を飲んでいれば大丈夫」という患者さんの安心感は、時に命取りになります。リスク評価を医師任せにせず、薬剤師として「目に見えるサイン」をどう拾い上げ、介入すべきか。その重要性を痛感した出来事でした。
</div>

抗血栓薬（抗血小板薬や抗凝固薬）を服用する患者さんにとって、血栓予防は最優先事項ですが、一方で避けて通れないのが<strong>「出血リスク」</strong>の管理です。今回は、明日からの服薬指導ですぐに活用できる評価ポイントを整理します。

---

## 📉 1. 出血リスク評価の基本「HAS-BLEDスコア」を知る

循環器内科などでよく活用される<strong>HAS-BLEDスコア</strong>。これは心房細動患者における出血リスクを予測するための指標ですが、薬剤師がチェックリストとして活用するのにも最適です。

| 項目 | 内容 |
| :--- | :--- |
| <strong>H</strong>ypertension | 高血圧（収縮期160mmHg超） |
| <strong>A</strong>bnormal renal/liver function | 腎・肝機能異常 |
| <strong>S</strong>troke | 脳卒中の既往 |
| <strong>B</strong>leeding | 出血既往・出血素因 |
| <strong>L</strong>abile INR | INRが不安定（ワーファリン服用時） |
| <strong>E</strong>lderly | 高齢（65歳超） |
| <strong>D</strong>rugs/Alcohol | 併用薬（NSAIDs等）・多量飲酒 |

※合計スコアが高いほど出血リスクが高まります。特に「Drugs（併用薬）」は薬剤師が介入できる最大のポイントです。

---

## 💊 2. 見逃してはいけない！出血リスクを高める併用薬

患者さんが何気なく飲んでいる「あのお薬」が、出血リスクを増大させているかもしれません。

<div style="display: grid; grid-template-columns: repeat(1, 1fr); gap: 1rem; margin: 1.5rem 0;">
  <div style="background-color: #fff1f2; padding: 1rem; border-radius: 8px; border: 1px solid #fda4af;">
    <div style="font-weight: bold; color: #9f1239; margin-bottom: 0.5rem;">🚨 特に注意が必要な併用薬</div>
    <ul style="color: #881337; font-size: 0.9em; padding-left: 1.2rem;">
      <li><b>NSAIDs（ロキソプロフェン等）</b>: 血小板凝集抑制作用により出血リスクが大幅に上昇。</li>
      <li><b>SSRI/SNRI</b>: セロトニン再取り込み阻害により、血小板の機能に影響を与え出血しやすくなる。</li>
      <li><b>抗真菌薬（アゾール系）</b>: CYP3A4を阻害し、DOACの血中濃度を上昇させる可能性がある。</li>
      <li><b>健康食品・サプリメント</b>: EPA/DHA製剤、イチョウ葉エキス等（抗血小板作用を持つ場合がある）。</li>
    </ul>
  </div>

## 🔍 3. 現場で使える！出血の「レッドフラッグサイン」

服薬指導時に、「変わりありませんか？」と聞くのはもうやめましょう。具体的に確認すべき項目をリスト化しました。

1. <strong>歯磨き時の出血</strong>: 以前より長引いていないか？
2. <strong>皮下出血</strong>: ぶつけた覚えがないのに青あざ（紫斑）が増えていないか？
3. <strong>便の色</strong>: タール便（黒色便）は出ていないか？（上部消化管出血のサイン）
4. <strong>尿の色</strong>: 血尿はないか？

<div style="background-color: #fefce8; border: 1px solid #fde047; padding: 1rem; border-radius: 8px; color: #854d0e;">
  <strong>💡 薬剤師のワンポイントアドバイス</strong><br>
  「出血したときに、すぐに病院へ行くべき目安」を具体的に伝えましょう。「普段より明らかに血が止まりにくい」「便が黒い」「激しい頭痛が続く（脳出血の可能性）」があれば、自己判断で休薬せず、すぐに主治医に連絡するよう指導することが重要です。
</div>

## 📝 4. まとめ：薬剤師が果たすべき役割

抗血栓薬服用患者の<strong>出血リスク評価</strong>は、単なるスコアリングではなく、患者さんの「日常生活」の中に隠れています。

* <strong>併用薬のチェック</strong>: OTCやサプリメントまで聞き出す。
* <strong>腎機能の変化</strong>: 加齢や脱水に伴うクレアチニンクリアランス（Ccr）の変化をフォローする。
* <strong>教育的介入</strong>: 「出血したらすぐ報告」という関係性を構築する。

「お薬の数を減らす」ことと同じくらい、「出血事故を防ぐ」ことは薬剤師にとって重要な使命です。今回の評価項目を、ぜひ次回の服薬指導から活用してみてください。
