# 疲れやすくなったら？薬剤師が見逃したくない病態とは

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  数年前、漢方薬の「補中益気湯」を定期処方されている患者さんが、薬局で「先生、いくら飲んでも疲れが取れないのよね」とこぼされました。私は「季節の変わり目ですし、お疲れが溜まっているのでしょうね」と、一般的な励ましの言葉をかけて聞き流してしまいました。<br><br>
  しかし数週間後、その患者さんがひどい動悸と倦怠感で緊急搬送されたという知らせを受けました。診断は「甲状腺機能亢進症（バセドウ病）」。漢方で疲れを癒やそうとしていたその背景には、医学的な病態が隠れていたのです。もっと早くに「体重の増減や便通の変化はありませんか？」と一言確認できていれば……。この経験は、今でも私の服薬指導の教訓になっています。
</div>

患者さんの「疲れやすい」「だるい」という訴えは、日常の薬局業務で非常に多く聞く内容です。しかし、これが単なる疲労なのか、それとも疾患のサインなのかを見極めることは、薬剤師の専門性が問われる場面でもあります。今回は、<strong>「見逃したくない疲労の裏側」</strong>について整理します。

---

## 🔍 1. 「ただの疲れ」じゃない？薬剤師が注目すべきレッドフラッグ

問診時に、単なる疲労感と疾患を区別するための「切り分け」が重要です。特に以下の症状を伴う場合、服薬指導時に一度立ち止まって確認してみてください。

| 確認項目 | 疑われる病態のサイン |
| :--- | :--- |
| <strong>体重の変化</strong> | 急激な増減は甲状腺機能異常や糖尿病の可能性 |
| <strong>便通・発汗</strong> | 下痢・多汗は亢進症、便秘・寒がりは低下症を示唆 |
| <strong>顔色・爪</strong> | 蒼白や爪の変形（匙状爪）は鉄欠乏性貧血を示唆 |
| <strong>睡眠の質</strong> | 日中の強い眠気は睡眠時無呼吸症候群の疑い |
| <strong>薬剤の副作用</strong> | β遮断薬、降圧薬、ステロイドの離脱などを確認 |

---

## 🧬 2. 実務で見逃せない！「疲れ」を訴える代表疾患

### ① 甲状腺機能異常症
甲状腺ホルモンの分泌異常は、代謝に直接影響するため「倦怠感」として現れます。

- <strong>甲状腺機能亢進症（バセドウ病など）</strong>: 代謝が過剰になり、「動悸」「多汗」「体重減少」「手指の震え」を伴う疲れやすさ。
- <strong>甲状腺機能低下症（橋本病など）</strong>: 代謝が低下し、「無気力」「体重増加」「寒がり」「便秘」を伴う疲れやすさ。

### ② 鉄欠乏性貧血
特に月経のある女性や、胃腸の疾患がある高齢者に多く見られます。
- <strong>サイン</strong>: 「階段を登った時の息切れ」「立ちくらみ」。
- <strong>薬剤師の視点</strong>: 市販の胃薬や鎮痛剤を長期連用していないか確認してください。慢性的な胃粘膜障害による出血が貧血の原因となっているケースがあります。

### ③ 糖尿病・高血糖
血糖コントロールが不良になると、エネルギーを効率よく利用できず慢性的な倦怠感が生じます。
- <strong>サイン</strong>: 「口渇」「多尿」「体重の減少」。

## ⚠️ 4. 「薬」が疲れを引き起こしている可能性も

病気ではなく、現在飲んでいるお薬が「疲れやすさ」の原因になっていることもあります。

- <strong>β遮断薬（メインテート等）</strong>: 心拍数を抑えるため、運動時の倦怠感が出やすい。
- <strong>抗ヒスタミン薬</strong>: 眠気やだるさが残ることがある。
- <strong>ステロイド</strong>: 急な休薬により副腎不全に近い状態（倦怠感）を招くことがある。

もし患者さんが「薬を飲み始めてからなんとなく調子が悪い」と訴えたら、副作用の可能性を考慮し、必ず医師へのフィードバックや処方提案を検討してください。

## 🌿 5. まとめ：薬剤師ができる「ゲートキーパー」の役割

「疲れやすい」という訴えは、患者さんから発せられる<strong>微細なSOS</strong>です。
私たち薬剤師は、薬を渡すだけでなく、その背後にある体の変調に気づける「ゲートキーパー（門番）」でなければなりません。

1. <strong>話を遮らずにまずは受け止める</strong>
2. <strong>体重・汗・便通などの「代謝のサイン」をそれとなく聞く</strong>
3. <strong>副作用の可能性を常に頭の片隅に置く</strong>

もし、これまでの診察の範囲外と思われるような変化を感じたら、「一度、内科や内分泌内科で血液検査を含めた相談をしてみてはいかがですか？」と、受診を勧奨することも大切な薬剤師の職能です。

あなたのその一言が、患者さんの深刻な疾患を早期に見つけるきっかけになるかもしれません。明日からの服薬指導で、ぜひ意識してみてくださいね。
