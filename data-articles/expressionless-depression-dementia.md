# 🧠 表情が少なくなったら？うつ症状や認知症を考える

<div style="background-color: #f8fafc; border-left: 4px solid #0284c7; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の在宅訪問エピソード】</strong><br>
  在宅訪問で長年担当している80代の男性患者さん。これまでは私が訪問すると「おぉ、先生よく来たね」と穏やかな笑顔で迎えてくれていました。しかし、ある時期から声をかけてもうつむいたままで、明らかに<strong>表情が少ない</strong>状態に。「お父さん、最近全然笑わなくなって、物忘れも増えた気がするんです。認知症でしょうか…」とご家族から不安そうに相談を受けました。<br><br>
  処方箋を見ると、数ヶ月前に追加された「胃薬（H2受容体拮抗薬）」と、他科で処方された「マイスリー」が。私はこの表情の変化が、単なる認知症の進行ではなく、別の要因ではないかと考えアセスメントを始めました。

薬局の窓口や在宅医療の現場で、高齢患者さんの「表情の変化」にハッとさせられる瞬間はありませんか？
<strong>「表情が少ない 高齢者」</strong>に接した際、私たちが真っ先に思い浮かべるのは「うつ病」や「認知症」ですが、実はその背景には複雑な病態や<strong>薬剤性の副作用</strong>が絡み合っていることがあります。

今回は、薬剤師が実務で役立つ<strong>「認知症とうつの見分け方」</strong>、<strong>「アパシー（意欲低下）」へのアプローチ</strong>、そして注意すべき<strong>「薬剤性パーキンソニズム（仮面様顔貌）」</strong>について、臨床的な視点から詳しく解説します！

### 😔 ① 老年期うつ病：「表情が乏しく、不安げ」
若年層のうつ病とは異なり、高齢者のうつ病は「気分の落ち込み」を言葉で訴えず、<strong>「体がだるい」「眠れない」「お腹が痛い」などの身体症状（仮面うつ病）</strong>として現れることが非常に多いです。不安や焦りから眉間にしわが寄り、表情が暗く硬くなるのが特徴です。

### 🧠 ② 認知症：「戸惑いによる表情の減少」
アルツハイマー型認知症などでは、周囲の会話が理解しづらくなり、<strong>「どう振る舞っていいかわからない」という戸惑い</strong>から表情が少なくなります。また、脳血管性認知症では、感情がコントロールできなくなる「感情失禁」や、感情表現が乏しくなる「感情鈍麻」が見られます。

### 🌀 ③ アパシー（意欲低下）：「無関心・無表情」
<strong>アパシー（意欲低下）は認知症</strong>に伴う重要な非認知機能障害（BPSD）の一つです。「うつ病」と混同されがちですが、本人は悲しんだり苦しんだりしておらず、ただ単に<strong>「あらゆることに関心や意欲が湧かない（無関心）」</strong>状態です。そのため、悲壮感のない、からっぽな無表情さが特徴です。

## ⚖️ 2. 実務で使える！「認知症 うつ 見分け方」

「物忘れ」や「無表情」が、うつ病によるもの（仮性認知症）なのか、本当の認知症なのか。薬剤師の窓口でのアプローチ、いわゆる<strong>「薬剤師 服薬指導」</strong>時に役立つアセスメントの視点を表にまとめました。

### 📊 認知症と老年期うつの鑑別シート

| 項目 | 認知症（初期） | 老年期うつ病（仮性認知症） |
| :--- | :--- | :--- |
| <strong>発症の仕掛け</strong> | 緩やかに、いつの間にか始まる | 比較的急速に、契機（喪失体験など）が明確 |
| <strong>本人の訴え</strong> | 物忘れを隠そうとする（「大丈夫」と繕う） | 自ら「物忘れがひどい」「頭が働かない」と嘆く |
| <strong>質問への応答</strong> | 的外れな回答、または取り繕う（辻褄を合わせる） | <strong>「わかりません」</strong>とすぐに諦める・回答を拒む |
| <strong>表情・態度</strong> | 穏やか、時に取り繕いの笑顔。後半に無表情化 | <strong>不安、焦燥、苦渋に満ちた表情</strong>。姿勢も前屈み |
| <strong>一日の変動</strong> | 夕方から夜間にかけて悪化しやすい（夜間せん妄） | 朝方に調子が悪く、夕方に改善することが多い（日内変動） |
| <strong>抗認知症薬の反応</strong> | 認知機能低下の進行を緩やかにする | 効果が薄い（抗うつ薬で表情や記憶力が劇的に改善） |

## 💊 3. 見落としてはならない「仮面様顔貌 薬剤性」の恐怖

患者さんの「表情が少なくなった」原因として、薬剤師が最も輝くアセスメント領域は<strong>「薬剤性パーキンソニズム」</strong>や<strong>「薬剤性うつ・アパシー」</strong>の発見です。

パーキンソン病の典型症状である「仮面様顔貌（まばたきが少なく、仮面をかぶったように無表情になること）」が、実は処方薬によって引き起こされているケースが多々あります。

### 🚫 薬剤性パーキンソニズム・表情減少を引き起こしやすい主な薬剤

以下の薬剤を服用中、または増量後に「表情が少ない」と感じた場合は、副作用を疑う必要があります。

| 薬効分類 | 代表的な薬剤名 | 注意すべき機序と症状の特徴 |
| :--- | :--- | :--- |
| <strong>抗精神病薬</strong> | ハロペリドール、リスペリドンなど | ドパミンD₂受容体遮断作用による。高齢者では少量でも仮面様顔貌や動作緩慢が出やすい。 |
| <strong>消化管運動改善薬</strong> | イトプリド、メトクロプラミドなど | ドパミン遮断作用を持つため、胃薬として長期漫然投与されている高齢者でパーキンソニズムを惹起。 |
| <strong>H2受容体拮抗薬</strong> | ファモチジン、ラニチジンなど | 中枢移行性があり、高齢者で認知機能低下、うつ状態、せん妄を引き起こすことがある。 |
| <strong>抗不安薬・睡眠薬</strong> | ベンゾジアゼピン系全般（トリアゾラム、エチゾラムなど） | 筋弛緩作用や中枢抑制作用により、表情筋が弛緩し、アパシー様のぼんやりした表情になる。 |
| <strong>抗ヒスタミン薬</strong> | 第一世代（ヒドロキシジンなど）、一部の総合感冒薬 | 強い抗コリン作用・中枢移行性により、認知機能を低下させ、無気力・無表情を招く。 |

<div style="background-color: #fef2f2; border: 1px solid #fca5a5; padding: 1rem; border-radius: 8px; color: #991b1b; margin: 1.5rem 0;">
  <strong>🚨 薬剤師のファインプレー！</strong><br>
  内科で処方されている「胃の薬（メトクロプラミド）」や「眠気覚まし・抗アレルギー薬」が原因で、パーキンソン病のような無表情（仮面様顔貌）になっている高齢患者さんは少なくありません。処方カスケード（副作用に対してさらに別の薬が処方されること）を防ぐためにも、新規処方や増量時期と「表情の変化」のタイムラインを必ず確認しましょう。
</div>

## 🗣️ 4. 薬剤師が実践する「受診勧奨・疑義照会」のスマートな伝え方

「表情が少ない」と感じる患者さんをアセスメントした結果、うつ病、認知症、あるいは薬剤性の疑いが高まった場合、どのように医師や家族にアプローチすべきでしょうか。

### 👥 ご家族へのアプローチ（服薬指導時）
ご家族に対して「認知症かもしれません」「うつ病です」と直接病名を出すのは、家族をパニックに陥れたり、拒絶反応を招いたりするため避けるべきです。

> <strong>❌ 避けるべき表現：</strong>
> 「お父さん、表情が全然ないので認知症かうつの可能性があります。精神科に行ってください」

> <strong>🟢 望ましい表現（気づきを促す）：</strong>
> 「最近、お父さんのご様子で、以前に比べて『おしゃべりが減ったな』とか『笑うことが少なくなったな』と感じる瞬間はありますか？ 実は、今飲まれているお薬の影響や、少し気持ちが落ち込みやすくなっているサインのこともあるんです。一度、主治医の先生に普段のご様子を伝えてみませんか？」

### 🩺 医師への提案・疑義照会の文面テンプレート
医師へ情報提供や処方提案（トレーシングレポート）を行う際は、<strong>「具体的な行動観察」「お薬のタイムライン」</strong>を客観的に示すことが重要です。
