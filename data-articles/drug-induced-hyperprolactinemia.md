# 💊 乳房の張りが強くなったら？薬剤性を疑うポイント

<div style="background-color: #f8fafc; border-left: 4px solid #0284c7; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット・気付きエピソード】</strong><br>
  ある日、慢性的な胃もたれで「ナウゼリン（ドンペリドン）」を継続服用している30代の女性患者さんが来局されました。<br>
  お薬をお渡しする際、「最近、なんだか生理前でもないのに胸が張って痛むんです。婦人科に行くべきでしょうか…」と不安そうに相談を受けました。<br><br>
  当時の私は「乳房の張り＝婦人科疾患」と直結させてしまい、すぐに受診を勧めようとしました。しかし、ふと患者さんの薬歴が頭をよぎりました。「まてよ、ドンペリドンを数ヶ月飲んでいるな…」と。<br><br>
  医師への確認（疑義照会）を経て、胃薬を別のタイプ（モサプリド）に変更してもらったところ、数週間で乳房の張りはすっきりと消失。患者さんから「お薬のせいだったなんて驚きました！あのとき気づいてくれてありがとう」と感謝されました。

女性にとって「乳房の張りや痛み」は非常にデリケートで不安な症状です。しかし実は、医療現場でよく使われる身近な薬が原因で起こる<strong>「薬剤性高プロラクチン血症」</strong>が隠れているケースが少なくありません。

今回は、薬剤師が知っておくべき<strong>「乳房の張り 薬剤性」を疑うための鑑別ポイントと服薬指導のコツ</strong>について徹底解説します！

## 🔬 1. なぜ薬で乳房が張る？薬剤性高プロラクチン血症のメカニズム

なぜ、精神科のお薬や胃腸薬を飲むと乳房が張ったり、母乳が出たりするのでしょうか？
その鍵を握るのが、脳の下垂体から分泌されるホルモン<strong>「プロラクチン（催乳ホルモン）」</strong>です。

通常、脳内では<strong>ドパミン</strong>が下垂体に作用して、プロラクチンの分泌を「ブロック（抑制）」しています。

<div style="background-color: #eff6ff; padding: 1.25rem; border-radius: 8px; border: 1px solid #bfdbfe; margin: 1.5rem 0;">
  <div style="font-weight: bold; color: #1e3a8a; margin-bottom: 0.5rem; font-size: 1.1em;">💡 プロラクチン分泌抑制のしくみ</div>
  <ul style="margin: 0; padding-left: 1.2rem; color: #1e40af;">
    <li style="margin-bottom: 0.5rem;"><strong>通常時</strong>：ドパミン ➡️ 下垂体のD2受容体を刺激 ➡️ プロラクチン分泌を<strong>抑制</strong></li>
    <li><strong>薬剤服用時</strong>：ドパミン拮抗薬 ➡️ D2受容体を遮断 ➡️ 抑制が外れ、プロラクチンが<strong>過剰分泌（高プロラクチン血症）</strong></li>
  </ul>
</div>

プロラクチンが過剰になると、授乳期でもないのに<strong>乳房の張り・痛み、乳汁分泌</strong>が起こるほか、性腺刺激ホルモンの分泌が抑えられるため<strong>月経不順（無月経）や不妊</strong>の原因にもなります。

## 🛠️ 4. 薬剤師から医師への代替薬（処方提案）のアプローチ

薬剤性が強く疑われる場合、医師に処方変更の提案（疑義照会・トレーシングレポートの提出）を行います。その際、<strong>「代替薬の提案」</strong>をセットで行うと非常にスムーズです。

### 🟢 胃腸薬・吐き気止めを変更する場合
D2受容体遮断作用を持たない、またはその作用が極めて弱い薬剤を提案します。

*   <strong>ドンペリドン（ナウゼリン）の代替</strong>
    *   👉 <strong>モサプリド（ガスモチン）</strong>：セロトニン5-HT4受容体刺激薬。プロラクチン上昇の心配がありません。
    *   👉 <strong>アコチアミド（アコファイド）</strong>：機能性ディスペプシア（FD）治療薬。アセチルコリンエステラーゼ阻害作用。
*   <strong>メトクロプラミド（プリンペラン）の代替（吐き気止め）</strong>
    *   👉 頓服程度の一時的な使用であれば様子を見ることも多いですが、継続が必要な場合は他系統への変更を検討します。

### 🔵 抗精神病薬を変更する場合
精神科領域では、原疾患（統合失調症など）のコントロールが最優先されるため、薬剤師の一存での変更提案は慎重に行うべきです。ただし、情報提供として以下の選択肢を提示できます。

*   <strong>リスペリドン等の代替</strong>
    *   👉 <strong>アリピプラゾール（エビリファイ）</strong>：ドパミン・システム・スタビライザー（部分作動薬）。ドパミンを完全に遮断しないため、<strong>プロラクチンを上昇させにくい（むしろ低下させることもある）</strong>特徴があります。
