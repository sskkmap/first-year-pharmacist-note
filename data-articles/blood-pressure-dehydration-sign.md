# 🩺 血圧変化から考える脱水のサイン

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード：その「血圧低下」、本当に薬の影響？】</strong><br>
  ある日の門前薬局。高血圧で降圧薬を服用している80代の患者さんが、「最近、ふらつきがあって血圧がいつもより15mmHgほど低いんです。薬を減らしたほうがいいでしょうか？」と来局されました。<br><br>
  単純に「血圧が低いなら減薬を検討しましょう」と安易に医師へ疑義照会していたら……実はその数日後、その患者さんは脱水と栄養不足で緊急入院することになったのです。後から振り返れば、皮膚の乾燥や舌の荒れ、脈拍のわずかな上昇など、脱水のサインはいくつも隠れていました。薬剤師として、血圧という「数値」の裏側にある「水分状態」を読み解く重要性を痛感した出来事でした。
</div>

循環器系の患者さんは、降圧薬の影響も相まって脱水に気づきにくいケースが多々あります。「血圧が下がった＝薬が効きすぎている」と決めつける前に、脱水を疑う視点を持つことが、重症化を防ぐ鍵となります。

---

## 📉 1. 血圧と脱水の相関関係：なぜ血圧が下がるのか？

脱水が進行すると、体内の循環血液量が減少します。最初は代償機転が働き、心拍数を上げて血圧を維持しようとしますが、限界を超えると以下のような変化が現れます。

### 脱水で血圧が変化するメカニズム
1. <strong>循環血液量の減少</strong>: 発汗や水分摂取不足により血管内のボリュームが減る。
2. <strong>心拍出量の低下</strong>: 戻ってくる血液が減るため、心臓が送り出す血液も減少する。
3. <strong>血圧の低下</strong>: 末梢血管の抵抗だけでは維持できなくなり、収縮期血圧が低下する。
4. <strong>起立性低血圧の増悪</strong>: 立ち上がった際に血液が下半身に溜まり、脳への血流が一時的に遮断されてふらつきが生じる。

特に降圧薬（ACE阻害薬、ARB、利尿薬など）を服用している患者さんは、体内の水分調整機能が薬によって制御されているため、<strong>脱水による低血圧が起こりやすい</strong>リスクを常に抱えています。

## 📋 2. 薬剤師がチェックすべき「脱水のサイン」一覧

血圧の変動以外にも、現場で確認できる「脱水のサイン」は多岐にわたります。以下の表を参考に、患者さんへのカウンセリング時にさりげなく観察してみましょう。

| 観察項目 | 脱水のサイン（異常所見） | 薬剤師の確認ポイント |
| :--- | :--- | :--- |
| <strong>血圧</strong> | 普段より収縮期血圧が10〜20mmHg低い | 低下した時期と体調不良のタイミング |
| <strong>脈拍</strong> | 普段より10回/分以上速い（頻脈） | 脈が弱く速い、いわゆる「細脈」でないか |
| <strong>舌・口腔</strong> | 舌の乾燥、縦皺（たてじわ）がある | 「口の中が乾きませんか？」と聴取 |
| <strong>皮膚</strong> | 皮膚の緊張度（ツルゴール）低下 | 手背の皮膚をつまみ、戻りが遅いか確認 |
| <strong>尿</strong> | 尿量が減っている、尿の色が濃い | 排尿回数や色の変化を聞く |
| <strong>自覚症状</strong> | ふらつき、立ちくらみ、こむら返り | 起立時の症状の有無を聴取 |

### ⚠ 注意すべき薬物群
* <strong>利尿薬（ラシックス、サムスカ等）</strong>: 水分排泄を促すため、脱水時は医師へ減量相談が必要なケースが多いです。
* <strong>ACE阻害薬/ARB</strong>: 脱水で腎血流量が低下している場合、急激な腎機能悪化のリスクがあります。

## 🏁 まとめ：数値の裏を読む「薬剤師の観察眼」

血圧が低いとき、それは単に「薬が効いている」だけではないかもしれません。
脱水は、<strong>「血圧の低下」「頻脈」「ふらつき」</strong>という3つのキーワードで察知できます。

普段の血圧手帳を見て、「あれ？最近低めだな」と感じたときには、ぜひ患者さんの<strong>「舌の潤い」や「皮膚の張り」、「尿の状態」</strong>にも目を向けてみてください。私たち薬剤師のちょっとした気づきが、患者さんの急変を防ぐ最後の砦になります。

次回の服薬指導からは、ぜひ「血圧の数値」だけでなく、<strong>「全身の水分状態」</strong>を評価する視点を取り入れてみてくださいね！
