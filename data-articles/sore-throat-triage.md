# のどの痛みが増えたら？受診勧奨のタイミングとは

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード：その「のどの痛み」は風邪ですか？】</strong><br>
  薬局のOTCカウンターで、「風邪っぽくてのどが痛い。一番強い薬をちょうだい」と訴える患者さんがいらっしゃいました。ルーチン通り炎症止めとトラネキサム酸を提案しようとした時、ふと見ると患者さんの喉元が少し赤く腫れており、何より声が少しこもっている（含み声）ことに気づきました。<br><br>
  「念のためにお熱はありますか？」と聞くと「38度くらい」とのこと。慌てて「まずは喉の奥を鏡で見てください。扁桃が極端に腫れていませんか？」と確認すると、明らかに通常の急性咽頭炎とは異なる様子でした。結果的にその方は急性扁桃炎から周囲に膿が溜まる「扁桃周囲膿瘍」の疑いがあり、すぐの受診勧奨で入院となりました。あの時、薬だけで済ませていたらと思うと今でも背筋が凍ります。
</div>

日々、多くの「のどの痛み」を訴える患者さんと接する中で、<strong>「どこまでがセルフケアで、どこからが医療機関の領分か」</strong>という線引きは、薬剤師にとって最も重要なスキルの一つです。今回は、その判断基準を整理します。

---

## 🚩 1. 絶対に見逃してはいけない「レッドフラッグ」

以下の症状がある場合は、薬の提案よりも「まず受診すること」を優先しましょう。

| チェック項目 | 受診を急ぐべき理由 |
| :--- | :--- |
| <strong>呼吸困難・息苦しさ</strong> | 気道狭窄や喉頭浮腫による窒息のリスク |
| <strong>嚥下障害（唾液が飲み込めない）</strong> | 扁桃周囲膿瘍や喉頭蓋炎の疑い |
| <strong>片側だけの強い腫れ</strong> | 深頸部感染症へ進行する可能性が高い |
| <strong>開口障害（口が開きにくい）</strong> | 炎症が咀嚼筋へ波及しているサイン |
| <strong>声の変化（含み声）</strong> | 扁桃の腫大による気道確保の困難 |

これらがある場合は、のんびりと市販薬を勧めている時間は残されていません。

### 💡 薬剤師の視点：溶連菌感染症を疑うなら
溶連菌は放置すると合併症（腎炎やリウマチ熱）のリスクがあります。<strong>「咳・鼻水がないのに喉だけ異常に痛い」</strong>という場合は、セルフケアの提案よりも、医療機関での検査を促すのが薬剤師としての誠実な対応です。

## 💊 3. 受診までの「対症療法」と「指導のポイント」

受診勧奨を行う場合でも、痛みが強ければ生活の質が著しく低下します。受診までの間にできる「ホームケア」を指導しましょう。

1.  <strong>室内の保湿（湿度50〜60%）</strong>: 乾燥は喉の粘膜の防御機能を著しく下げます。
2.  <strong>水分のこまめな摂取</strong>: 喉の粘膜を乾燥させないことが、痛みの緩和に直結します。
3.  <strong>刺激物を避ける</strong>: 熱すぎるもの、辛いもの、酸っぱいものは炎症を増悪させます。
4.  <strong>トラネキサム酸の活用（OTC）</strong>: 受診まで数時間〜1日ある場合、炎症を抑えるトラネキサム酸製剤の短期的使用は有効です。

## 📝 まとめ：薬剤師ができる「ゲートキーパー」の役割

薬剤師の役割は、単に薬を販売・調剤することではありません。<strong>「その症状が、今すぐ病院に行くべきものなのか、様子を見て良いものなのか」を判断するトリアージ</strong>こそが、地域医療における薬剤師の真価です。

- <strong>症状が長引く（1週間以上）</strong>
- <strong>抗生物質を希望するが、明らかにウイルス性</strong>
- <strong>痛みが日に日に強くなっている</strong>

こういった場合は、患者さんの希望通りに薬を出すのではなく、「なぜ今、病院での診察が必要なのか」を言語化して伝えること。その丁寧な説明こそが、患者さんを重大な疾患から守る第一歩になります。

「のどの痛み」というありふれた主訴の裏に隠れたリスクを見逃さないよう、今日からもう一歩踏み込んだヒアリングを心がけてみてくださいね。
