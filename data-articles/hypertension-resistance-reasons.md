# 💊 降圧薬を飲んでいるのに血圧が高い理由とは？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  「先生、この血圧の薬、全然効かないよ！毎日飲んでるのに、家で測ると150を超えちゃうんだ。」<br><br>
  ある日、ARB、CCB、利尿薬の3剤を服用中の患者さんからそう訴えられました。処方医も「おかしいな…」と増量を検討。しかし、よくよくお話を伺ってみると、実は「血圧が高いと感じた時だけ飲んでいた」という事実が判明したのです。<br><br>
  薬理学的な効果を疑う前に、まず私たち薬剤師が確認すべき「血圧が下がらない理由」が他にもたくさん隠れていました。

降圧薬を3剤以上（利尿薬を含む）併用しても目標血圧に達しない状態を<strong>「治療抵抗性高血圧」</strong>と呼びます。今回は、現場でよく遭遇する「血圧が下がらない背景」を深掘りしていきましょう。

## 💊 2. 薬剤師がチェックすべき「血圧を上げる薬剤・嗜好品」

併用薬やサプリメントが原因で血圧が上昇していることがあります。お薬手帳のチェックで見逃さないようにしましょう。

| 分類 | 代表的な薬剤・物質 | 血圧上昇のメカニズム |
| :--- | :--- | :--- |
| <strong>解熱鎮痛薬</strong> | NSAIDs（ロキソプロフェン等） | 腎血管拡張物質（PG）の抑制による塩分・水分貯留 |
| <strong>漢方薬</strong> | 甘草（グリチルリチン） | 偽性アルドステロン症によるカリウム排泄、血圧上昇 |
| <strong>感冒薬</strong> | エフェドリン、プソイドエフェドリン | 交感神経刺激作用による血管収縮 |
| <strong>精神神経用薬</strong> | 三環系抗うつ薬、一部の抗精神病薬 | 交感神経系の賦活 |
| <strong>ステロイド</strong> | プレドニゾロン等 | ミネラルコルチコイド作用による水・電解質保持 |
| <strong>嗜好品</strong> | 過度の飲酒、多量のカフェイン | 血管収縮、交感神経の緊張 |

## 🧬 3. 「二次性高血圧」の可能性を考慮する

もしアドヒアランスも良好で、薬剤性も否定できる場合、背景に疾患が隠れている「二次性高血圧」かもしれません。以下の疾患が有名です。

### 🏔️ ① 睡眠時無呼吸症候群 (SAS)
- <strong>特徴</strong>: 夜間の低酸素状態で交感神経が興奮。
- <strong>ヒアリングポイント</strong>: 「夜中に何度も目が覚める」「昼間の眠気が強い」「いびきを指摘される」など。

### 🧪 ② 原発性アルドステロン症 (PA)
- <strong>特徴</strong>: アルドステロンが過剰分泌され、カリウムが排出。
- <strong>ヒアリングポイント</strong>: <strong>「低カリウム血症」</strong>の既往がないか。血液検査結果があれば要チェックです。

### 💧 ③ 腎実質性・腎血管性高血圧
- <strong>特徴</strong>: 腎機能の低下や、腎動脈の狭窄によるもの。
- <strong>ヒアリングポイント</strong>: eGFRの急激な低下や、急に血圧コントロールが悪化した場合に疑います。

## 🍽️ 4. 生活習慣の「落とし穴」を再確認

「お薬さえ飲んでいれば大丈夫」という誤解を解くのも薬剤師の仕事です。

## 🩺 5. 薬剤師から医師への疑義照会・提案のポイント

血圧が高い原因が見えてきたら、次は処方設計への介入です。

1.  <strong>配合剤への変更提案</strong>:
    錠数が多いことによる飲み忘れが原因なら、ARB/CCB配合剤や、さらに利尿薬を加えた3剤配合剤（例：ミコンビ、トラクリア等）への変更が有効です。
2.  <strong>利尿薬の追加検討</strong>:
    体液貯留傾向（むくみ等）がある場合、少量のチアジド系利尿薬の追加が治療抵抗性を打破する鍵になることが多いです。
3.  <strong>服用時間の検討</strong>:
    早朝高血圧がある場合、一部の薬剤を夕食後や寝る前にシフトすることでコントロールが改善する場合があります。
