# 🚽 トイレに行く回数が減ったら？脱水を疑うポイント

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  ある日の門前薬局でのこと。「最近、なんだかトイレに行く回数が減って…」と、80代の患者さんが相談に来られました。「もしかして、お薬の副作用かしら？」と心配そうです。<br><br>
  詳しく話を聞くと、持病の高血圧で利尿薬を服用されており、最近の暑さで食欲も少し落ちているとのこと。私はてっきり排尿トラブルかと思いましたが、お薬手帳を確認すると、特に排尿に関わる薬の変更はありませんでした。念のため手の甲の皮膚をつまんでみると、戻りが明らかに遅い……。これは、典型的な<strong>「脱水症状」</strong>のサインだったのです。
</div>

「トイレの回数が減る＝腎機能が悪化した？」と焦る前に、まずは水分バランスの崩れを疑う視点が、私たち薬剤師には求められています。今回は、服薬指導の現場で活用できる「脱水のサイン」とアセスメントのポイントを整理します。

---

## 🔍 1. なぜ「排尿回数」が脱水の指標になるのか？

健康な成人は、通常1日に4〜8回程度排尿します。しかし、高齢者の場合、<strong>「喉の渇きを感じにくい（口渇中枢の低下）」</strong>という身体的特徴があるため、自覚症状がないまま脱水が進むことが少なくありません。

脱水が進むと身体は「貴重な水分」を逃がさないように腎臓で再吸収を強めます。その結果、<strong>「尿量が減る（＝トイレの回数が減る）」</strong>という現象が起こります。

### 🚨 見逃してはいけない「脱水のサイン」

排尿回数の減少を訴える患者さんが来局した際、以下のチェックリストを確認してみてください。

| チェック項目 | 脱水時の状態 |
| :--- | :--- |
| <strong>手の甲の皮膚</strong> | つまんで離した時、すぐに戻らない（テント状に残る） |
| <strong>口の中</strong> | 唾液が少なく、舌が乾燥してネバネバしている |
| <strong>腋下（わきの下）</strong> | 指を入れても湿り気がない、乾いている |
| <strong>脈拍・血圧</strong> | 脈が速くなる（頻脈）、血圧が普段より低め（起立性低血圧） |
| <strong>体重</strong> | 短期間で急激に減少している |

## 📉 2. 薬剤師が確認すべき「リスク要因」

患者さんが服用しているお薬の中には、脱水を引き起こしやすくするものがあります。特に以下の薬剤を服用中の患者さんには、意識的な水分補給のアドバイスが必須です。

      <li><b>緩下剤（下剤）</b> → 下痢による水分喪失の可能性がある</li>

## 💊 3. 患者さんへのアドバイス：どう水分を摂るか？

「お水をたくさん飲んでください」だけでは、心不全や腎機能低下のある患者さんには不十分、あるいは禁忌となる場合があります。以下のステップでアドバイスを行うのが現場でのコツです。

### ステップ①：現在の「飲水習慣」を聞く
「お茶やコーヒー、お水を含めて、1日にどのくらい飲まれていますか？」と具体的に聞いてみましょう。特にカフェイン入りの飲料は利尿作用があるため、水分補給には適さない場合があることを伝えます。

### ステップ②：経口補水液（OS-1など）の提案
単なる水よりも、電解質が含まれた経口補水液の方が効率的に吸収されます。
- <strong>「喉が渇いたと感じる前に、コップ一杯の水分を摂る」</strong>
- <strong>「食事の汁物や果物からも水分を補えることを伝える」</strong>

---

## ⚠️ 4. 「いつ病院に行くべきか？」の判断基準

薬剤師として、以下のような症状が見られる場合は、<strong>直ちに医師へ相談（受診勧奨）</strong>するよう促してください。

*   <strong>意識がぼーっとしている、反応が鈍い</strong>（高度脱水の疑い）
*   <strong>尿の色が非常に濃い（濃縮尿）</strong>
*   <strong>熱がないのに立ちくらみやめまいがひどい</strong>
*   <strong>体重が数日で2〜3kg以上減った</strong>

---

## 🌿 まとめ：些細な変化をキャッチする専門家として

「トイレの回数が減った」という患者さんの主訴は、薬剤調整の必要性を見つけるための貴重なヒントです。

今回お話しした患者さんの場合も、利尿薬の用量調節は医師の判断が必要ですが、<strong>「脱水対策として、日中の水分摂取をもう少し意識しましょう」</strong>と指導し、翌週の受診時にその旨を医師に伝えることで、安全な治療の継続に繋げることができました。

私たち薬剤師の強みは、<strong>「薬の副作用」と「身体の変化」を照らし合わせて、いち早くリスクに気づけること</strong>です。ぜひ、次回の服薬指導から「お口の中の乾燥」や「皮膚の張り」をチェックする習慣を取り入れてみてください。
