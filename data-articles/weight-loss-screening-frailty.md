# BMIが減ったら？フレイルや悪性疾患を疑うポイント

<div style="background-color: #f0f9ff; border-left: 4px solid #0284c7; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場でのリアルな気づき】</strong><br>
  ある高齢の患者さんが「最近、ズボンが緩くなった気がする」とおっしゃいました。当時は「食欲が落ちているなら、少し高カロリーな栄養補助食品でも勧めてみようかな」と軽く考えていました。しかし、念のためお薬手帳を確認すると、数ヶ月前からPPI（胃酸抑制薬）が開始されており、さらに直近ではHbA1cが急激に改善（低下）していることが分かりました。<br><br>
  「おかしい」と思い、食事量や喉のつかえ感について詳しく伺うと、胃カメラの予約を控えているとのこと。単なる加齢によるフレイルではなく、背景に悪性疾患が隠れていた症例です。この時、もし私が栄養指導だけで済ませていたら、発見が遅れていたかもしれません。
</div>

体重減少は、私たち薬剤師が患者さんのわずかな変化から「レッドフラッグ（危険信号）」を読み取るための重要な指標です。今回は、BMI低下を起点にフレイルや疾患をどう見分けるか、実務的なポイントを整理します。

---

## 🔍 1. 体重減少を見逃さないための判断基準

「なんとなく痩せた」という言葉を鵜呑みにするのは危険です。医学的に注意すべき体重減少の目安を確認しましょう。

| 指標 | 警戒すべき基準 |
| :--- | :--- |
| <strong>体重減少率</strong> | 1ヶ月で5%以上、または6ヶ月で10%以上の減少 |
| <strong>BMI値</strong> | 18.5未満（低栄養の可能性大） |
| <strong>フレイル指標</strong> | 握力の低下、疲労感、歩行速度の低下、活動量の低下 |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1.5rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 薬剤師が確認すべき「体重減少」の原因フロー</strong>
  <ul style="margin: 0.5rem 0 0 0; padding-left: 1.2rem;">
    <li><strong>医原性：</strong>新規開始・増量された薬による副作用はないか？（例：メトホルミン、SGLT2阻害薬、甲状腺ホルモン薬）</li>
    <li><strong>精神・口腔：</strong>うつ症状、噛み合わせの不具合、義歯の不適合はないか？</li>
    <li><strong>器質的疾患：</strong>嚥下障害、腹痛、便通異常、出血はないか？</li>
  </ul>
</div>

## ⚠️ 2. 「ただの加齢」か「疾患」かを見分けるヒント

高齢者の体重減少は「フレイル」と片付けられがちですが、以下のような項目をチェックし、<strong>「非意図的な体重減少」</strong>かどうかを見極めることが重要です。

### 🚨 悪性疾患を疑うべきサイン
*   <strong>貧血の進行:</strong> 検査値（Hb, MCV）の変動。
*   <strong>炎症反応:</strong> CRPの上昇傾向。
*   <strong>局所症状:</strong> 嚥下困難、血便、排便習慣の変化、しこり。
*   <strong>夜間の盗汗（寝汗）:</strong> 悪性リンパ腫などで見られることがあります。

### 👵 フレイル・低栄養を疑うサイン
*   <strong>食事の質:</strong> 栄養が偏り、タンパク質不足が顕著（アルブミン値の低下）。
*   <strong>生活環境:</strong> 一人暮らしの開始、買い物や料理の困難さ。
*   <strong>多剤服用:</strong> 食欲低下を誘発する薬剤の重複や相互作用。

---

## 💊 3. 薬剤師ができる「疑義照会」と「観察」の具体例

私たちが服薬指導の中でできるアプローチをまとめました。

### 薬物療法の見直し
もし患者さんが痩せてきたと感じたら、まずはお薬手帳を見直しましょう。特に以下の薬剤は体重減少の原因になり得ます。

*   <strong>SGLT2阻害薬:</strong> 体重減少は主作用ですが、フレイル患者では過度な脱水や低栄養のリスクになります。
*   <strong>メトホルミン:</strong> 高齢者では消化器症状（食欲不振・下痢）により体重が落ちることがあります。
*   <strong>コリンエステラーゼ阻害薬（ドネペジル等）:</strong> 認知症治療薬による食欲低下は非常に多い副作用の一つです。

### 医師への報告テンプレート（例）
> 「◯◯様ですが、過去3ヶ月で体重が約5kg（6%）減少しております。食事量の減少も訴えておられ、現在服用中の◯◯（薬剤名）の副作用、あるいは潜在的な疾患の可能性も含め、一度血液検査や栄養状態の評価を検討いただけないでしょうか。」

---

## ✨ 4. まとめ：薬剤師は「変化の番人」

BMIの低下は、患者さんからの<strong>「助けて」という静かなサイン</strong>です。

1.  <strong>数字の変化を記録する:</strong> 体重・BMIの推移は可能な限り手帳にメモする。
2.  <strong>変化の「理由」を問いかける:</strong> 「食事を変えた？」「運動を始めた？」と尋ね、もし「何もしていないのに」という答えなら要注意。
3.  <strong>多角的に評価する:</strong> 検査値、お薬の副作用、生活環境をトータルで見る。

私たち薬剤師が、些細な変化を見逃さず医師へ橋渡しをすることで、深刻な疾患の早期発見や、適切なフレイル対策につながります。明日からの服薬指導で、ぜひ患者さんの「見た目」や「服装の変化」にも目を向けてみてくださいね。
