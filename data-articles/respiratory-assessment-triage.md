# 呼吸状態の変化から受診勧奨を判断するには？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  以前、継続して吸入薬を処方されている患者さんから、「最近、薬を吸っても息苦しさがすぐには取れなくて……」と相談を受けたことがあります。その時は「手技に問題がないか」を真っ先に確認し、吸入指導を重点的に行いました。<br><br>
  しかし翌週、その患者さんは呼吸困難が悪化し救急搬送されました。実は心不全による肺水腫が隠れていたのです。肺の「音」を直接聞けない薬剤師として、言葉の裏にある「呼吸状態の変化」をどう捉えるべきだったのか、痛烈な反省が残りました。
</div>

呼吸器疾患を持つ患者さんは、日々の服薬指導が患者さんの異変を察知する貴重な機会となります。今回は、私たち薬剤師が服薬指導の現場で活用できる「呼吸器系トリアージの考え方」について解説します。

---

## 🔍 1. 呼吸困難を訴える患者さんへのチェックリスト

患者さんが「息が苦しい」と言ったとき、まずは以下の要素を確認しましょう。特に<strong>「安静時」か「労作時」か</strong>の区別は極めて重要です。

| チェック項目 | 緊急度（高）のサイン |
| :--- | :--- |
| <strong>呼吸の速さ</strong> | 安静時でも肩で息をしている（頻呼吸） |
| <strong>呼吸の様子</strong> | 横になれない（起坐呼吸）、会話が途切れる |
| <strong>SpO2値</strong> | 基準値（常用値）から3-4%以上の低下 |
| <strong>随伴症状</strong> | 胸痛、チアノーゼ、冷や汗、意識の混濁 |
| <strong>発症様式</strong> | 急激に悪化したか（数分〜数時間） |

<div style="background-color: #fff1f2; border: 1px solid #fda4af; padding: 1rem; border-radius: 8px; color: #881337; margin-top: 1rem;">
  <strong>🚨 今すぐ受診を促すべきレッドフラッグ</strong><br>
  ・「横になると苦しいから座っていたい（起坐呼吸）」<br>
  ・「一息で文章を話せない（失語性呼吸）」<br>
  ・「唇や爪の色が少し青っぽく見える（チアノーゼ）」<br>
  これらは重篤な呼吸不全や心不全の可能性があるため、その場で直ちに医療機関への連絡を促してください。
</div>

---

## 🔬 2. 「呼吸状態」の観察ポイント

聴診器がなくても、観察できるポイントはたくさんあります。以下の項目を意識して観察してみましょう。

### 💬 言葉の区切り方に注目する
会話がスムーズかどうかは、肺の予備能を測る簡便な指標です。「息が苦しくて、一度に長く話せない」状態は、かなり肺に負担がかかっています。

### 🫀 浮腫（むくみ）の確認
「呼吸が苦しい」という訴えが、実は肺そのものではなく<strong>心臓由来（肺うっ血）</strong>であるケースが多々あります。
*   足のスネを押してみて、指の跡が残る（圧痕性浮腫）
*   最近、体重が急激に増えていないか
これらを確認し、呼吸症状と合わせて医師へ情報提供することで、より早期の診断につながります。

## ⚖️ 3. 受診勧奨の判断フロー

薬剤師の判断基準として、以下のステップで対応を検討してください。

## 🌿 まとめ：患者さんの「言葉の裏側」を聴く

「呼吸が苦しい」という患者さんの訴えに対し、手技の確認だけで終わらせず、その背後にある<strong>全身状態の変化</strong>を見極めることが薬剤師の重要な役割です。

1.  <strong>普段との比較</strong>: 患者さんの「いつもの息切れ」と「今の苦しさ」の違いを聞き出す。
2.  <strong>身体観察</strong>: 浮腫や表情、話し方など、視覚的・聴覚的情報を逃さない。
3.  <strong>迅速なアクション</strong>: 少しでも違和感があれば、自己判断で様子を見させず、主治医へ繋ぐ勇気を持つ。

呼吸器疾患は刻一刻と変化します。私たち薬剤師の判断が、患者さんの予後を大きく変えるきっかけになることを常に意識しておきましょう。
