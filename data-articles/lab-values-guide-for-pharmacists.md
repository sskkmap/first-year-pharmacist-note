# 🩺 検査値が読めると、服薬指導が変わる！

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験：腎機能編】</strong><br>
  ある日、80代の小柄な女性患者さんに「ガスター錠20mg 1日2回」の処方箋が届きました。電子薬歴の検査値を見ると、eGFRは 55.0 mL/min/1.73m²。「少し低いけど、常用量で問題ないかな」と監査を済ませようとした時、ふと手が止まりました。<br><br>
  「この方、体重が35kgしかない…。もしや？」と思い、Cockcroft-Gault式でCcr（クレアチニン・クリアランス）を計算し直してみると、数値はなんと <b>25.0 mL/min</b> 近くまで低下していたのです。<br><br>
  もしそのままお渡ししていたら、過量投与によるせん妄などの副作用を引き起こしていたかもしれません。検査値の「数字の裏側」を読む大切さを痛感した出来事でした。
</div>

薬剤師の皆さんは、検査値を見て「基準値内だからOK」と機械的に判断していませんか？
今回は、実務で特に重要な<strong>腎機能・肝機能</strong>を中心に、薬剤師が押さえておくべき検査値の深掘りポイントを整理していきましょう！

## 🧪 1. 腎機能の評価：eGFRとCCr、どっちを使うべき？

薬剤師を最も悩ませるのが、腎機能の評価指標です。現在、多くの医療機関でeGFRが採用されていますが、薬剤の投与設計においては注意が必要です。

## 🔬 2. 肝機能の評価：ダメージか、機能不全か？

肝機能の数値は「肝細胞が壊れているのか（逸脱酵素）」と「肝臓がしっかり働いているのか（合成能）」を分けて考えるのがコツです。

### 🧬 主要な肝機能検査値の読み方

<div style="display: grid; grid-template-columns: repeat(2, 1fr); gap: 1rem; margin: 1.5rem 0;">
  <div style="background-color: #fef2f2; padding: 1rem; border-radius: 8px; border: 1px solid #fecdd3;">
    <div style="font-weight: bold; color: #9f1239; margin-bottom: 0.5rem; border-bottom: 2px solid #fecdd3; padding-bottom: 0.25rem;">肝細胞のダメージ（逸脱酵素）</div>
    <ul style="margin: 0; padding-left: 1.2rem; color: #be123c; font-size: 0.9em;">
      <li><b>AST/ALT</b>：肝細胞が壊れると上昇。ALTの方が肝特異性が高い。</li>
      <li><b>γ-GTP/ALP</b>：胆道系の指標。薬物性肝障害（胆汁うっ滞型）で上昇しやすい。</li>
    </ul>
  </div>
  <div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe;">
    <div style="font-weight: bold; color: #1e3a8a; margin-bottom: 0.5rem; border-bottom: 2px solid #bfdbfe; padding-bottom: 0.25rem;">肝臓の「働く力」（合成能など）</div>
    <ul style="margin: 0; padding-left: 1.2rem; color: #1e40af; font-size: 0.9em;">
      <li><b>Alb（アルブミン）</b>：肝臓で作られるタンパク質。低下は予備能低下を示す。</li>
      <li><b>PT（プロトロンビン時間）</b>：凝固因子の産生能力。低下すると出血しやすくなる。</li>
    </ul>
  </div>
</div>

### 🚨 薬物性肝障害（DILI）を疑うタイミング
薬剤師として特に注意したいのは、新しく薬が始まった後の数値変化です。
- <strong>ALTが基準値上限の3倍を超えた場合</strong>
- <strong>ALPが基準値上限の2倍を超えた場合</strong>
これらが見られたら、直近で開始した薬剤がないか、サプリメントの服用はないか、速やかに確認しましょう。

## 🚩 4. まとめ：検査値は「点」ではなく「線」で見る

検査値を使いこなすために最も大切なのは、<strong>「推移（トレンド）」</strong>を見ることです。
一度の数値が基準値内であっても、急激に上昇・低下している場合は、体の中で何かが起きているサインかもしれません。

<div style="background-color: #fef2f2; border: 1px solid #fca5a5; padding: 1rem; border-radius: 8px; color: #991b1b; margin-top: 1rem;">
  <strong>🚨 要注意！</strong><br>
  HbA1cが急激に下がっている場合、一見「改善」に見えますが、実は<strong>腎機能が悪化したことで低血糖リスクが高まっている</strong>可能性もあります。複数の検査値を組み合わせて多角的に判断しましょう。
</div>

冒頭のエピソードの続きですが、医師に「体重換算でのCCrが低下しているため、ガスターを10mgへ減量、または1日1回への変更はいかがでしょうか」と疑義照会したところ、「そこまで見てくれてありがとう、助かるよ」と快諾いただけました。

患者さんの安全を守り、医師から信頼される薬剤師になるために。
明日からの業務では、ぜひ<strong>「この患者さんのこの数値、今の処方で大丈夫かな？」</strong>と一歩踏み込んで考えてみてくださいね！
