# 階段で休む回数が増えたら？薬剤師が確認したいこと

<div style="background-color: #fff7ed; border-left: 4px solid #f97316; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場でヒヤリとした一言】</strong><br>
  ある日の服薬指導中、継続して血圧の薬を飲まれている高齢の患者さんが、「先生、最近足が弱ってね。昔は一気に登れた駅の階段も、今は途中で2回も休まなきゃいけないんだよ」と仰いました。<br><br>
  その時は私も「年齢のせいですね、無理なさらないでくださいね」と笑顔で返してしまいました。しかし、翌週その患者さんは呼吸困難で緊急入院。原因は心不全の悪化でした。あの時、ただの老化だと思い込まず、心不全の兆候を疑うべきだった……と深く後悔しました。
</div>

薬剤師の日常業務において、患者さんのちょっとした「活動量の変化」を見逃さないことは、疾患の重症化を防ぐための重要な防波堤となります。今回は、<strong>「階段で休む回数が増えた」という訴えに対して、薬剤師が疑うべき疾患と、明日から使える確認ポイント</strong>を整理します。

## 🔍 なぜ「階段」が重要なのか？

「息切れ」や「疲労感」を訴える患者さんは多いですが、特に<strong>「階段」という具体的なシチュエーション</strong>は、身体への負荷が客観的に示されるため、心機能の予備能を評価するバロメーターになります。

心不全や貧血、あるいは薬剤性の浮腫など、原因は多岐にわたります。まずは以下の観点で整理してみましょう。

| 確認すべきポイント | 具体的な質問例 | 疑われるリスク |
| :--- | :--- | :--- |
| <strong>負荷のタイミング</strong> | どのくらいの段数で息が切れますか？ | 心不全（NYHA分類） |
| <strong>浮腫の有無</strong> | 最近、靴がきつい・靴下の跡が残りませんか？ | 体液貯留（うっ血） |
| <strong>寝る時の姿勢</strong> | 横になると苦しくて、枕を高くしますか？ | 起座呼吸（心不全） |
| <strong>随伴症状</strong> | 咳や痰は出ていませんか？ | 肺うっ血・薬剤性咳嗽 |

## 💊 薬剤師が注目すべき「服薬中の薬」

患者さんの症状が、疾患の進行ではなく<strong>「お薬の影響」</strong>である可能性も考慮しなければなりません。特に以下の薬剤は注意が必要です。

### 1. 浮腫を引き起こしやすい薬剤
*   <strong>カルシウム拮抗薬（アムロジピンなど）</strong>: 用量依存的な下肢浮腫が起こりやすく、体重増加や活動量低下を招くことがあります。
*   <strong>NSAIDs</strong>: 水分・塩分の貯留を促進し、心不全を増悪させるリスクがあります。

### 2. 息切れ・咳を引き起こしやすい薬剤
*   <strong>β遮断薬</strong>: 喘息持ちの患者さんでは気管支収縮を引き起こし、息切れの原因になります。
*   <strong>ACE阻害薬</strong>: 空咳が続くことで、心不全による息切れと鑑別が難しくなることがあります。

## 🎯 結論：現場でのアクションプラン

患者さんが「階段で休む回数が増えた」と言ったら、まずは以下のステップを踏んでください。

1.  <strong>「いつから」かつ「どの程度」増えたのか</strong>を明確にする。（急激な変化なら要注意）
2.  <strong>体重変化を尋ねる</strong>。（「最近、体重が増えていませんか？」は心不全の早期発見に極めて有効です）
3.  <strong>むくみの有無を確認する</strong>。（靴下の跡や、すねを指で押して戻りを確認）
4.  <strong>疑義照会・処方医への報告</strong>。
    *   これらの兆候があれば、速やかに主治医へ情報をフィードバックしましょう。単なる「加齢」というバイアスを外すことが、患者さんの命を守る第一歩です。

「単なる疲れ」という言葉の裏には、重大な病気が隠れていることがあります。私たちの専門知識で、そのサインを早期に拾い上げていきましょう。
